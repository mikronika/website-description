Обеспечьте надёжное энергоснабжение вашего микрокомпьютера Raspberry Pi 4 оригинальным источником питания от Raspberry Pi Foundation.

Видеообзор

Возможности Raspberry Pi 4

Общие сведения

На входе блока стандартная европейская вилка EU: не придётся искать переходник у метро. На выходе зарядника расположен кабель длиной 1,5 м с интерфейсом USB Type-C. Блок питания подойдёт не только для малинки, но и для других медиа продуктов с аналогичным разъёмом для подключения напряжения.

The Official Raspberry Pi USB-C выдаёт напряжение 5,1 Вольт с максимальным током 3 Ампера, что обеспечивает стабильное энергоснабжение устройства.

Комплектация

Характеристики

  • Входное напряжение: AC 220 В
  • Выходное напряжение: DC 5,1 В
  • Выходной ток: 3 A
  • Длина кабеля: 1,5 м
  • Выходной разъём: USB Type-C
  • Цвет: белый, черный
