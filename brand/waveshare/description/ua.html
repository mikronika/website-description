Компанія Waveshare — топовий виробник DIY-модулів для розробки електроніки, робототехніки та систем розумного будинку. Компанія відома унікальними платами розширення для Arduino та Raspberry Pi, великим вибором MIPI-камер, широким асортиментом дисплеїв та іншою електронікою. Waveshare відрізняється високою якістю, що робить її ідеальним вибором для мейкерів, ентузіастів та професіоналів.

Завдяки постійним інноваціям і увазі до деталей, Waveshare здобула довіру і повагу в спільноті розробників по всьому світу.
