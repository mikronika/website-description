Компания Raspberry Pi Foundation — британский производитель мини-компьютеров размером с кредитную карту. Эти компактные «малинки» по своим возможностям не уступают обычным ПК, а небольшие размеры делают их идеальными для использования во встраиваемых системах. Главной же отличительной чертой компьютеров Raspberry Pi против штатных ПК — являются пины ввода-вывода (GPIO), которые позволяют подключать к малинке кнопочки, лампочки и другие электронные компоненты.

Если вам нужен компактный бесшумный ПК — выбирайте Raspberry Pi.
