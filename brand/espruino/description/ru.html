Компания Espruino — британский производитель DIY-модулей и программного обеспечения для разработки и исполнения JavaScript-кода на привычных микроконтроллерах. Espruino предлагает прошивку и микроконтроллерные платы, позволяющие запускать JavaScript-код напрямую на устройстве без предварительной компиляции. Espruino идеально подходит для веб-разработчиков, которые хотят использовать знакомый язык JavaScript для создания проектов на микроконтроллерах.

Если вы хотите разрабатывать проекты на JavaScript для встраиваемых систем — выбирайте Espruino.
