Компанія Espruino — британський виробник DIY-модулів та програмного забезпечення для розробки й виконання JavaScript-коду на звичайних мікроконтролерах. Espruino пропонує прошивку та мікроконтролерні плати, що дозволяють запускати JavaScript-код безпосередньо на пристрої без попередньої компіляції. Espruino ідеально підходить для WEB-розробників, які хочуть використовувати знайому мову JavaScript для створення проєктів на мікроконтролерах.

Якщо ви хочете розробляти проєкти на JavaScript для вбудованих систем — обирайте Espruino.
