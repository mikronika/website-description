Подарите своему устройству на базе Arduino или Iskra проводною сеть. Ethernet Shield позволит выступить в роли полноценного сетевого устройства: общаться с аналогичными устройствами, с обычными компьютерами, принтерами, сервисами в интернете и другими сетевыми ресурсами.

Видеообзор

Общие сведения

Используйте Ethernet Shield для создания проектов IoT «интернета вещей»:

  • управляйте устройствами через интернет;
  • отображайте данные онлайн-сервисов вроде Яндекс.Пробок и Гисметео;
  • запускайте небольшие веб-сервера.

Начинка

Плата основана на чипе Wiznet W5500, который поддерживает TCP и UDP-протоколы. Чип позволяет работать в открытом состоянии до 8 соединений, а также поддерживает работу по протоколам HTTP и HTTPS.

Подключение и программирование

Для соединения по локальной сети используйте патч-корд или витую пару. Один конец кабеля подключите к роутеру или хабу, второй — к Ethernet Shield.

Распиновка модуля соответствует форм-фактору Arduino Uno. Для коммуникации с контроллером используется шина SPI через ICSP-контакты:MOSI, MISO, SCK, а также 10-й в качестве CS.

Альтернативы

На микроконтроллерной плате тяжело реализовывать обработку XML, JSON и раздавать страницы с динамическим содержимым — ограниченность в ресурсах даёт о себе знать. Поэтому если вам нужен проект с мощным веб-сервером или клиентом, присмотритесь к одноплатным компьютерам.

Характеристики

  • Ethernet-чип: Wiznet W5500
  • Рабочее напряжение: 3,3–5 В
  • Интерфейс управления: SPI
  • Поддерживаемые протоколы: TCP, UDP, ICMP, IPv4, ARP, IGMP, PPPoE
  • Скорость соединения: 10/100 Мбит
  • Количество сокетов: 8
  • Размер буфера: 32 КБ
  • Занимаемые пины: SPI (MISO, MOSI, SCK) и 10
  • 69×53 мм (RJ45 выступает на несколько мм)
