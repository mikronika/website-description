Управляющие платформы STM32 — платы для разработки на базе микроконтроллеров STM32 для создания собственного девайса. Контроллер служит для управления всей периферией и обрабатывает показания датчиков в реальном времени. Главная особенность платформ STM32 — это высокопроизводительные ARM-процессоры Cortex-M0/M3/M4/M7, которые дополнены большим количеством памяти и уймой интерфейсов ввода-вывода.
