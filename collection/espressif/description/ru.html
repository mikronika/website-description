Управляющие платформы на чипах ESP8266 и ESP32 — главная деталь в электронном устройстве для создания собственного девайса. Контроллер служит для управления всей периферией и обрабатывает показания датчиков в реальном времени. Главная особенность платформ ESP8266 / ESP32 — встроенный беспроводной модуль Wi-Fi.
