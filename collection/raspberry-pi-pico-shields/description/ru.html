Модули Pico Shields служат для расширения дополнительных возможностей контроллеров Raspberry Pi Pico. Платы расширения научат контроллеры Raspberry Pi Pico управлять двигателями, выходить в интернет, связываться с другими устройствами, работать с промышленным оборудованием и т.д. Платы Pico Shields заточены специально для контроллеров Raspberry Pi Pico, устанавливаются методом бутерброда без пайки и проводов — справится даже новичок!
