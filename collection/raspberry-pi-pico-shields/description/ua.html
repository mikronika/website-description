Модулі Pico Shields для розширення додаткових можливостей контролерів Raspberry Pi Pico. Плати розширення навчать контролери Raspberry Pi Pico керувати двигунами, виходити в інтернет, зв'язуватися з іншими пристроями, працювати з промисловим обладнанням тощо. Плати Pico Shields заточені спеціально для контролерів Raspberry Pi Pico, встановлюються методом бутерброду без паяння та дротів.
