Управляющая платформа Arduino — это электронный мозг для создания собственного девайса. Контроллер служит для управления всей периферией и обрабатывает показания датчиков в реальном времени. Платформа Ардуино завоевала популярность, потому что подходит для тех, кто не обладает знаниями в электронике.
