Модули Arduino Shields служат для расширения дополнительных возможностей контроллеров Arduino. Платы расширения научат Arduino управлять двигателями, выходить в интернет, связываться с другими устройствами, работать с промышленным оборудованием и т.д. Платы Arduino Shields заточены специально для контроллеров Arduino, устанавливаются методом бутерброда без пайки и проводов — справится даже новичок!
