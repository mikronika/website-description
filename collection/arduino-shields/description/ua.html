Модулі Arduino Shields потрібні для розширення додаткових можливостей контролерів Arduino. Плати розширення навчать Arduino керувати двигунами, виходити в інтернет, зв'язуватися з іншими пристроями, працювати з промисловим обладнанням тощо. Плати Arduino Shields заточені спеціально для контролерів Arduino, встановлюються методом бутерброду без паяння та дротів.
