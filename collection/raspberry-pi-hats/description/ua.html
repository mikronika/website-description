Модулі Raspberry Pi HATs для розширення додаткових можливостей комп'ютерів Raspberry Pi. Плати розширення навчать одноплатники Raspberry Pi керувати двигунами, зв'язуватися з іншими пристроями, працювати з промисловим обладнанням тощо. Плати Raspberry Pi HATs заточені спеціально для комп'ютерів Raspberry Pi, встановлюються методом бутерброду без паяння та дротів.
