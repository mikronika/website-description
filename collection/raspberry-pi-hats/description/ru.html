Модули Raspberry Pi HATs для расширения дополнительных возможностей компьютеров Raspberry Pi. Платы расширения научат одноплатники Raspberry Pi управлять двигателями, связываться с другими устройствами, работать с промышленным оборудованием и т.д. Платы Raspberry Pi HATs заточены специально для компьютеров Raspberry Pi, устанавливаются методом бутерброда без пайки и проводов.
