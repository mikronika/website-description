Дисплеї OLED для візуального відображення інформації з Arduino, Raspberry Pi, Espruino, micro:bit та інших контролерів. Дисплеї OLED (Organic Light Emitting Diode) виконані на матриці органічних світлодіодів, які самостійно випромінюють світло при проходженні через них електричного струму. Технологія ОLED перевершує технологію LCD за якістю, насиченістю та яскравістю зображення. Однак виготовлення матриць OLED коштує дорожче, тому технологія LCD ще міцно тримається на ринку.
