Дисплеи OLED для визуального отображения информации с Arduino, Raspberry Pi, Espruino, micro:bit и других контроллеров. Дисплеи OLED (Organic Light Emitting Diode) выполнены на матрице из органических светодиодов, которые самостоятельно испускают свет при прохождении через них электрического тока. Технология ОLED превосходит технологию LCD по качеству, сочности и насыщенности картинки. Однако изготовления матриц OLED стоит дороже, поэтому технология LCD ещё крепко держится на рынке.
