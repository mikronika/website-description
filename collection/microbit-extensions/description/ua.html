Модулі Micro:bit Extension потрібні для розширення додаткових можливостей контролерів BBC micro:bit. Плати розширення навчать контролери micro:bit керувати двигунами, виходити в інтернет, зв'язуватися з іншими пристроями, працювати з промисловим обладнанням тощо. Плати Micro:bit Extension заточені спеціально для контролерів BBC micro:bit, встановлюються без паяння та дротів.
