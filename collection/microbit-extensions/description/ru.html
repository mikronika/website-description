Модули Micro:bit Extension служат для расширения дополнительных возможностей контроллеров BBC micro:bit. Платы расширения научат контроллеры micro:bit управлять двигателями, выходить в интернет, связываться с другими устройствами, работать с промышленным оборудованием и т.д. Платы Micro:bit Extension заточены специально для контроллеров BBC micro:bit, устанавливаются без пайки и проводов — справится даже новичок!
