Платы из линейки Troyka модулей для расширения дополнительных возможностей Arduino, Raspberry Pi и других контроллеров. Экосистема Troyka — унифицированный формат для начинающих и опытных мейкеров, чтобы быстро реализовать свои замыслы, не путаться в многообразии интерфейсов и придать устройству эстетический вид.
