Raspberry Pi — популярный одноплатный мини-компьютер по кличке «малина» размером с кредитую карту. Малинка обладает достаточно мощной начинкой, чтобы заменить настольный ПК начального уровня. Компьютеры Raspberry Pi работают на Linux и поддерживают USB-устройства, дисплеи, камеры и другую периферию.
