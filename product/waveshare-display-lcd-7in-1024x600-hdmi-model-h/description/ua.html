Загальна інформація

Використовуйте кольоровий дисплей Waveshare LCD 7” (1024×600 / HDMI / Model H) для виведення зображення з Raspberry Pi, ігрових консолей, персональних комп’ютерів та інших пристроїв із HDMI-виходом. Дисплейний модуль стане в нагоді для перегляду медіафайлів, проходження відеоігор або трансляції відео з камери. Вбудована ємнісна сенсорна панель дозволяє використовувати екран як планшет для зручного керування системою без клавіатури та миші.

Начинка

Дисплейний модуль Waveshare LCD 7” Model H виконаний на матриці TFT IPS з діагоналлю 7 дюймів, роздільною здатністю 1024×600 пікселів, кутом огляду до 170° і відображенням 262144 кольорів. Ємнісна сенсорна панель підтримує до 5 одночасних дотиків.

Екран оснащений вбудованою світлодіодною підсвіткою, що забезпечує видимість зображення за будь-яких умов освітлення. У дисплеї є вбудоване меню для налаштування яскравості, контрастності, рівня підсвітки та інших параметрів зображення. На модулі передбачений аудіовихід 3,5 мм для підключення навушників та роз'єм JST з підсиленням звукового сигналу для зовнішньої акустики.

Підключення

На платі дисплея розташовані порти HDMI та VGA для отримання відеосигналу від зовнішніх пристроїв, а також роз'єм USB Micro для передачі даних з сенсорної панелі.

Живлення

Модуль живиться через порт USB Micro напругою 5 вольт зі споживанням струму до 900 мА. Живлення можна отримати через USB-порт від пристрою джерела відео або від зовнішнього блока живлення.
