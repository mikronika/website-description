Общие сведения

Используйте цветной дисплей Waveshare LCD 2,8” (320×240 / SPI / Model A) для вывода изображения с Raspberry Pi, Orange Pi, Rock Pi и других одноплатных ПК со стандартным GPIO-разъёмом на 40 контактов. Дисплейный модуль пригодится для вывода изображений, анимаций, графиков и диаграмм. Встроенная резистивная сенсорная панель позволяет задействовать экран в роли планшета для удобного управления системой без клавиатуры и мыши.

Начинка

Дисплейный модуль Waveshare LCD 2,8” Model A выполнен на матрице TFT TN с диагональю 2,8 дюймов, разрешением 320×240 пикселей и отображением 262144 цветов. Резистивная сенсорная панель реагирует на любое нажатие, включая пальцы в перчатках и стилусы без специального покрытия.

Экран оснащён встроенной светодиодной подсветкой, обеспечивающей видимость изображения при любых условиях освещённости. Сбоку дисплея расположены четыре пользовательские кнопки, на которые можно назначить фиксацию вспомогательных событий.

Подключение

Дисплей предусмотрен для установки на миникомпьютеры форм-фактора Raspberry Pi через GPIO-колодку без пайки и проводов. Для получения видеосигнала используется интерфейс SPI с дополнительными контактами для передачи данных с сенсорной панели.

Питание

Плата с дисплеем питается от Raspberry Pi через GPIO-разъём напряжением 5 вольт с потреблением тока до 200 мА.
