Загальна інформація

Використовуйте кольоровий дисплей Waveshare LCD 2,8” (320×240 / SPI / Model A) для виведення зображення з Raspberry Pi, Orange Pi, Rock Pi та інших одноплатних ПК зі стандартним GPIO-роз'ємом на 40 контактів. Дисплейний модуль стане в нагоді для виведення зображень, анімацій, графіків і діаграм.

Вбудована резистивна сенсорна панель дозволяє використовувати екран як планшет для зручного керування системою без клавіатури та миші. Резистивний тачскрін потрібно програмно відкалібрувати для більш точної та коректної роботи.

Начинка

Дисплейний модуль Waveshare LCD 2,8” Model A виконаний на матриці TFT TN з діагоналлю 2,8 дюйма, роздільною здатністю 320×240 пікселів і відображенням 262144 кольорів. Резистивна сенсорна панель реагує на будь-яке натискання, зокрема на дотики пальців у рукавичках та стилусів без спеціального покриття.

Екран оснащений вбудованою світлодіодною підсвіткою, що забезпечує видимість зображення за будь-яких умов освітлення. Збоку дисплея розташовані чотири користувацькі кнопки, на які можна призначити фіксацію допоміжних подій.

Підключення

Дисплей передбачений для встановлення на мінікомп'ютери форм-фактора Raspberry Pi через GPIO-колодку без пайки та проводів. Для отримання відеосигналу використовується інтерфейс SPI з додатковими контактами для передачі даних з сенсорної панелі.

Живлення

Плата з дисплеєм живиться від Raspberry Pi через GPIO-роз'єм напругою 5 вольт зі споживанням струму до 200 мА.
