Раскройте мультимедийные возможности Raspberry Pi с цветным сенсорным LCD-дисплеем Waveshare. Просматривайте фото и видео, играйте в ретро-игры или транслируйте живое изображение с RPi-камеры. С помощью экранного модуля можно собрать планшет для индикации показаний датчиков и управления освещением в системе умного дома или организовать мониторинг видеонаблюдения. Модуль подключается к Raspberry Pi методом бутерброда — просто установите дисплей сверху. После чего соедините обе платы специальным коннектором по HDMI. Экран также может использоваться в качестве монитора для игровых консолей и ПК.

LCD-дисплей Waveshare выполнен на IPS-матрице с диагональю 3,5 дюйма, разрешением 480×320 пикселей и углом обзора до 160°. Программно можно увеличить разрешение до 1920×1080 пикселей. Экран оснащён резистивным сенсором и распознаёт нажатия пальцем или стилусом.

Через собственное меню дисплея можно настроить яркость, контрастность, уровень подсветки и другие параметры изображения. На модуле предусмотрен аудиовыход 3,5 мм для подключения наушников.

Дисплейный модуль Waveshare использует интерфейс HDMI для передачи изображения, а сенсорная панель подключается через SPI.
