Плата расширения Pimoroni Enviro:bit превратит контроллер BBC micro:bit в настоящую метеостанцию! Модуль нафарширован сенсорами для отслеживания климатических параметров, показаний освещённости и уровня шума.

Метеостанция Pimoroni Enviro:bit содержит три датчика для различных наблюдений:

  • Климатический сенсор BME280 — измеряет температуру, влажность воздуха и атмосферное давление.
  • Датчик цвета TCS3472 — позволяет оценить общую освещённость и определить цветовой оттенок объекта в падающем свете.
  • МЭМС-микрофон SPU0410HR5H-PB — выступает в роли датчика уровня шума.

Метеостанция Pimoroni Enviro:bit заточена для установки на контроллеры BBC micro:bit через краевой разъём без пайки и проводов. Для коммуникации используется программный интерфейс I²C и аналоговый сигнал.
