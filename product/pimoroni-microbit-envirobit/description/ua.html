Плата розширення Pimoroni Enviro:bit перетворить контролер BBC micro:bit на справжню метеостанцію! Модуль оснащений сенсорами для відстеження кліматичних параметрів, показників освітленості та рівня шуму.

Метеостанція Pimoroni Enviro:bit містить три датчики для різних спостережень:

  • Кліматичний сенсор BME280 — вимірює температуру, вологість повітря та атмосферний тиск.
  • Датчик кольору TCS3472 — дозволяє оцінити загальну освітленість і визначити колірний відтінок об'єкта у паданому світлі.
  • МЕМС-мікрофон SPU0410HR5H-PB — виконує функцію датчика рівня шуму.

Метеостанція Pimoroni Enviro:bit розроблена для встановлення на контролери BBC micro:bit через крайовий роз'єм без пайки та проводів. Для комунікації використовується програмний інтерфейс I²C та аналоговий сигнал.
