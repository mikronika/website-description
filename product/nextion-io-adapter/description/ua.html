Дисплеї Nextion — це не просто кольорові сенсорні екрани, а повноцінна універсальна система людино-машинного інтерфейсу (англ. Human-Machine Interface, HMI), сумісна з Arduino, Raspberry Pi, ESP та іншими контролерами.

Перехідник Nextion I/O допоможе розкрити весь потенціал розумних дисплеїв Nextion — забезпечить зручний доступ до 10 контактів HMI для підключення та управління різними DIY-модулями самостійно, тобто без додаткового зовнішнього контролера.

Nextion I/O підходить для лінійок дисплеїв Nextion лінійок Enhanced та Intelligent.
