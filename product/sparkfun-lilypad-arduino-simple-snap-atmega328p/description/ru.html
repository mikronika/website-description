Используйте контроллер SparkFun LilyPad Arduino Simple Snap в качестве электронного мозга для создания умной одежды. Управляющая плата поможет собрать скатерть с индикацией температуры еды, умную перчатку для управления девайсами с помощью жестов, футболку с мигающими светодиодами в такт музыке и другие девайсы в текстильном направлении.

Контроллер LilyPad Arduino Simple Snap выполнен в виде плоского круглого диска диаметром 50 мм. По внешнему диаметру на плате расположены 11 контактов в виде кнопок для прищёлкивания к одежде в заранее пришитую ответную часть. Кнопочное решение позволяет комфортно отсоединить устройство от одной вещи и прикрепить к другой без траты времени на пришивание.

Платформа выполнена на микроконтроллере ATmega328P c тактовой частотой 16 МГц и программируется на языке C++. Для загрузки прошивки на плате расположены контакты PLS-6 к которым понадобится подключить внешний USB – UART преобразователь.

Для питания контроллера используйте контакты PLS-6 с коммуникацией через USB – UART преобразователь или контакты-кнопки с метками «+» и «−». При подключении напряжения соблюдайте полярность, чтобы избежать повреждения устройства. На плате также расположен встроенный Li-Pol аккумулятор на 110 мАч для питания модуля в автономном режиме.
