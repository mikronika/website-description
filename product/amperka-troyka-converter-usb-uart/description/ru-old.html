Забудьте про вытаскивание микроконтроллера из Arduino Uno для прошивки плат без собственного USB-порта, например Iskra Mini или Troyka Wi-Fi. Компактный USB-UART преобразователь подружит любую электронику с USB-портом компьютера.

Видеообзор

Общие сведения

Модуль служит мостом между USB-портом компьютера и микроконтроллером. Это позволяет подключить к ПК любое устройство c последовательным интерфейсом UART.

Преобразователь уровней будет полезен для настройки и прошивки Wi-Fi модуля в наборе «Интернет вещей» — продолжение «Матрёшки».

USB-UART откроет доступ к тонкой настройке Bluetooth или GPS/GLONASS модулей через служебные команды, а также пригодится для перепрошивки самодельной Arduino.

Подключение

Подключайте модуль к компьютеру кабелем micro-USB. На преобразователя уровней выведено две пары Troyka-контактов.

Нижняя группа

  • Сигнальный (TX) — цифровой выход USB-UART преобразователя. Используется для передачи данных из USB в микроконтроллер. Подключите к пину RX микроконтроллера;
  • Питание (V) — соедините с рабочим напряжением микроконтроллера;
  • Земля (G) — Соедините с пином GND микроконтроллера.

Верхняя группа

  • Сигнальный (TX) — Сигнальный (RX) — цифровой вход USB-UART преобразователя. Используется для приёма данных из микроконтроллера в USB. Подключите к пину TX микроконтроллера;
  • Сигнальный (R) — пин сброса.
  • 1× Плата модуль
  • 2× Трёхпроводной шлейф
  • 1× Джампер
  • Чип: CP2102
  • Входное напряжение питания: 5 В от USB
  • Выходное напряжение питания: 3,3 / 5 В
  • Напряжение логический уровней: 3,3 В
  • Память буфера: чтения 576 байт / записи 640 байт
  • EEPROM-память: 1 КБ для хранения параметров идентификации USB-устройства
  • Габариты: 25,4×25,4 мм
