Використовуйте сервопривід Feetech FS5113R для створення багатооборотних столів для 3D-вітрин, обертання різних механізмів з регулюванням швидкості та як колеса для мобільних роботів.

Сервомотор Feetech FS5113R здатний встановлювати і підтримувати швидкість обертання вала до 55 обертів на хвилину по годинниковій або проти годинникової стрілки без обмежень по куту повороту. Вбудований редуктор з металевих шестерень дозволяє розвивати приводу крутний момент до 13 кг·см. Серводвигун працює при живлені від 4,8 до 8,4 вольт з споживанням струму до 200 мА без навантаження та до 2 А в режимі блокування вала.

Для керування сервоприводом FS5113R необхідно подавати сигнал PWM/PDM — імпульси постійної частоти і змінної ширини. За обробку сигналу відповідає аналоговий блок управління, який інтерпретує ширину імпульсів для задання швидкості та напрямку обертання вала. Як бонус на сервомоторі виведено додатковий сигнальний дрот FB для зчитування поточної швидкості та напрямку вала сервоприводу.
