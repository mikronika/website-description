Використовуйте сервопривід Feetech FB5317M-360 для створення багатооборотних столів для 3D-вітрин, обертання різних механізмів з регулюванням швидкості та як колеса для мобільних роботів.

Сервомотор Feetech FB5317M-360 здатний встановлювати і підтримувати швидкість обертання валу до 60 обертів на хвилину за годинниковою або проти годинникової стрілки без обмежень по куту повороту. Вбудований редуктор з металевих шестерень дозволяє приводу досягнути крутний момент до 18 кг·см. Серводвигун працює при живленні від 4,8 до 8,4 вольт зі споживанням струму до 200 мА без навантаження та до 2,2 А в режимі блокування вала.

Для керування сервоприводом FB5317M-360 необхідно подавати сигнал PWM/PDM — імпульси постійної частоти та змінної ширини. За обробку сигналу відповідає цифровий блок управління на мікроконтролері, який інтерпретує ширину імпульсів для задання швидкості та напрямку обертання валу. Як бонус, на сервомоторі виведено додатковий сигнальний дрот FB для зчитування поточної швидкості та напрямку обертання валу сервоприводу.
