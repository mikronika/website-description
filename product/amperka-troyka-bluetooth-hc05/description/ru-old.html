Хотите подружить платформу Arduino с мобильным телефоном? Возьмите Troyka-модуль Bluetooth HC-05, который поможет управлять DIY-роботом или мониторить данные домашней метеостанции через смартфон. А ещё Bluetooth может организовать беспроводную сеть между двумя управляющими платами.

Видеообзор

Общие сведения

Мозгом Bluetooth выступает модуль HC-05 с чипом BC417 от Cambridge Silicon Radio.

Поставив на телефон или планшет одну из многочисленных программ для управления через bluetooth, вы превратите его в настоящий продвинутый джойстик, и ваш робот сможет послушно поворачивать вслед за поворотом смартфона. Связывайте ваши устройства по Bluetooth: опрашивайте датчики, посылайте команды, управляйте роботами без какого-либо дополнительного софта.

Модуль HC-05 выполнен по технологии Bluetooth v2 и не работает с операционной системой iOS. Если вы обладатель мобильного телефона Apple, используйте Bluetooth v4 (BLE).

Подключение

Bluetooth-модуль общаеться с управляющей электроникой по интерфейсу UART с одним дополнительным пином. Для подключения используйте две группы трёхконтактных разъёмов.

Первая группа

  • Сигнальный (TX) — цифровой выход Bluetooh-модуля. Используется для передачи данных в микроконтроллер. Подключите к пину RX микроконтроллера.
  • Питание (V) — соедините с рабочим напряжением микроконтроллера.
  • Земля (G) — соедините с пином GND микроконтроллера.

Вторая группа

  • Сигнальный (RX) — цифровой выход Bluetooth-модуля. Используется для приёма данных из микроконтроллера. Подключите к пину TX микроконтроллера.
  • Сигнальный (K) — используйте для перевода модуля в режим управления АТ-командами. Подключите к любому цифровому пину микроконтроллера.

При подключении к Arduino или Iskra используйте Troyka Shield. Чтобы полностью избавиться от проводов, возьмите Troyka Slot Shield.

Программирование

Для управляющих платформ модуль выглядит как «прозрачный» последовательный интерфейс. Для начала работы необходимо настроить Bluetooth в режиме AT-команд. После чего — сопряжатся с другими Bluetooth-девайсами.

Bluetooth HC-05 работает в двух режимах:

  • Slave — модуль ожидает подключения с других Bluetooth-устройств. Например другого Bluetooth или телефона.
  • Master — модуль сам инициирует подключение к выбранному Bluetooth-девайсу.
Если вы новичок в программировании микроконтроллеров, обратите внимание на образовательные наборы Матрёшка и Йодо.

Характеристики

  • Bluetooth: модуль HC-05 на базе чипа BC417
  • Рабочее напряжение: 3,3–5 В
  • Потребляемый ток: 50 мА
  • Радиус действия: до 10 метров
  • Режим работы: Master и Slave
  • Интерфейс беспроводной сети: Bluetooth
  • Интерфейс к управляющей плате: UART
  • Габариты: 50,8×25,4 мм
