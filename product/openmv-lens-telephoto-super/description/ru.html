Общие сведения

Объектив OpenMV Super Telephoto Lens для камер OpenMV с восьмикратным зумом и байонетом M12/0,5 мм позволяет снимать удалённые объекты, которые практически не видны для стандартного объектива. Телеобъектив идеально подходит для ситуаций, когда требуется исследовать объекты крупным планом. Например, для системы трекинга глазных зрачков без использования ИК-подсветки.

Комплектация

  • 1× Объектив OpenMV Super Telephoto Lens

Характеристики

  • Совместимость: камеры OpenMV
  • Байонет: M12/0,5 мм
  • Фокусное расстояние: 25 мм
  • Эквивалентный зум: 8×
  • Диафрагма: F2.0
  • Угол обзора по горизонтали: 9,1°
  • Угол обзора по вертикали: 6,8°
  • Размеры: 14×14×20 мм
  • Вес: 5 г
