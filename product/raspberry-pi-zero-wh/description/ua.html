Raspberry Pi Zero WH — повноцінний одноплатний мікрокомп'ютер розміром з iPod, стане гідною заміною ноутбука або ПК. На борту наймініатюрнішої з «малинок» розмістилися процесор Broadcom BCM2835, модуль оперативної пам'яті об'ємом 512 МБ, бездротовий модуль з Wi-Fi / Bluetooth, слот для microSD карти, роз'єм HDMI Mini, конектор для штатних камер CSI Mini та два порти USB Micro.
