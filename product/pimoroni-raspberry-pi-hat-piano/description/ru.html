Используйте плату расширения Pimoroni Piano HAT для превращения миникомпьютера Raspberry Pi в музыкальный синтезатор! Сенсорные клавиши позволяют воспроизводить ноты, запускать звуковые эффекты и управлять MIDI-сигналами.

Сенсорное пианино Pimoroni Piano HAT содержит одну октаву на 13 клавиш. На модуле также расположены три функциональные кнопки: повышения октавы, понижения октавы и смена инструмента в синтезаторе. На каждой клавише расположен белый светодиод для индикации нажатия или подсказки музыкальной партии.

Piano HAT может работать в трёх режимах:

  • Режим репетитора: для обучения новых мелодий. Встроенные светодиоды подскажут последовательность нажатия клавиш для исполнения музыкальной композиции.
  • Программная MIDI-клавиатура: для воздействия на компьютерные MIDI-синтезаторы. Играйте музыку на виртуальных инструментах с помощью софтовых синтезаторов.
  • Аппаратная MIDI-клавиатура: для воздействия на штатные MIDI-синтезаторы. Piano HAT посылает стандартные MIDI-команды для управления внешним музыкальным оборудованием.

Pimoroni Piano HAT предусмотрена для установки на миникомпьютеры Raspberry Pi через GPIO-разъём без пайки и проводов.
