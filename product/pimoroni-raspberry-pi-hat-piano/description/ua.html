Використовуйте плату розширення Pimoroni Piano HAT для перетворення мінікомп'ютера Raspberry Pi на музичний синтезатор! Сенсорні клавіші дозволяють відтворювати ноти, запускати звукові ефекти та керувати MIDI-сигналами.

Сенсорне піаніно Pimoroni Piano HAT містить одну октаву на 13 клавіш. На модулі також розташовані три функціональні кнопки: підвищення октави, зниження октави та зміна інструмента у синтезаторі. На кожній клавіші розташований білий світлодіод для індикації натискання або підказки музичної партії.

Piano HAT може працювати у трьох режимах:

  • Режим репетитора: для навчання новим мелодіям. Вбудовані світлодіоди підкажуть послідовність натискання клавіш для виконання музичної композиції.
  • Програмна MIDI-клавіатура: для впливу на комп'ютерні MIDI-синтезатори. Грайте музику на віртуальних інструментах за допомогою софтових синтезаторів.
  • Апаратна MIDI-клавіатура: для впливу на штатні MIDI-синтезатори. Piano HAT надсилає стандартні MIDI-команди для керування зовнішнім музичним обладнанням.

Pimoroni Piano HAT призначена для встановлення на мінікомп'ютери Raspberry Pi через GPIO-роз'єм без пайки та проводів.
