Pimoroni Scroll:bit — це світлодіодний дисплей для контролерів BBC micro:bit з роздільною здатністю 17×7 точок у вигляді 119 яскравих білих світлодіодів. Матричний екран стане в нагоді для виведення тексту, анімацій та примітивної графіки.

Матриця зі світлодіодів підключена до чіпа Holtek IS31FL3731, який виконує функцію драйвера між екраном та зовнішніми керуючими платами. Мікросхема також дозволяє регулювати рівень яскравості, інтенсивність освітлення та імітувати напівтони.

Світлодіодна матриця Pimoroni Scroll:bit розроблена для встановлення на контролери BBC micro:bit через крайовий роз'єм без пайки та проводів. Для комунікації використовується програмний інтерфейс I²C.
