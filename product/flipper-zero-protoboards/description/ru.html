Общие сведения

Создавайте собственные DIY-модули для Flipper Zero с помощью макетных плат под пайку.

В комплект входят одна большая и две малые платы для прототипирования. Также набор дополнен штырьковыми вилками для удобного подключения к GPIO-контактам.

Стандартный шаг между отверстиями 2,54 мм позволяет установить сюда компоненты с выводными контактами: микросхемы, резисторы, светодиоды и другие.

Комплектация

  • 1× Большая макетная плата
  • 2× Малая макетная плата
  • 3× Штыревая вилка (10 pin)
  • 3× Штыревая вилка (8 pin)

Характеристики

  • Совместимость: Flipper Zero
  • Количество контактов:
    • Большая макетная плата: 486
    • Малая макетная плата: 252
  • Шаг между контактами: 2,54 мм
