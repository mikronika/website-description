Мініатюрний комп'ютер Raspberry Pi 4 — це платформа з процесором Broadcom BCM2711 на 4 ядра по 1,5 ГГц, 2 ГБ RAM та модулями Wi-Fi / Bluetooth, який повноцінно замінить ПК або ноутбук. Платформа підійде не тільки для веб-серфінгу всесвітньої павутини та спілкування в соцмережах, але й стане «мозком» для робота чи розумного дому. На борту малини вже є чотири порти USB, два HDMI Micro виходи під монітори, бездротовий модуль Wi-Fi / Bluetooth та повноцінний Ethernet зі швидкістю до 1 Гбіт.
