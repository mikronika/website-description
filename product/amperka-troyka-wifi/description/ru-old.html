Подключайте свои гаджеты к соцсетям, добавляйте модули в IoT-облака и пользуйтесь API веб-сервисов для управления ими. Wi-Fi на модуле ESP-12 с чипом ESP8226EX откроет беспроводной доступ для любых проектов на Arduino и Iskra.

Видеообзор

Начинка

Troyka Wi-Fi основан на модуле ESP-12 с чипом ESP8266EX от компании Espressif.

Стандартная прошивка через АТ-команды

По умолчанию модуль настроен на работу через «AT-команды» по интерфейсу UART. Управляющая плата посылает команды — Wi-Fi модуль выполняет соответствующую операцию. Для коммуникации используйте библиотеки для облегчения написания кода программ. Они скрывают в себе все тонкости общения с модулем.

  • Библиотека для Arduino
  • Библиотека для Espruino

Wi-Fi модуль как самостоятельный контроллер

Модуль Troyka Wi-Fi не только беспроводной приёмопередатчик. Под металлической крышкой прячется целый микроконтроллер, который можно программировать на языке C++ через Arduino IDE или JavaScript через Espruino Web IDE.

Боллее подробно читайте техническую документацию на модуль Wi-Fi.

Подключение

Приёмопередатчик общается с управляющей электроникой по интерфейсу UART с одним дополнительным пином. Для подключения используйте две группы трёхконтактных разъёмов.

Первая группа

  • Сигнальный (TX) — цифровой выход Wi-Fi-модуля. Используется для передачи данных в микроконтроллер. Подключите к пину RX микроконтроллера.
  • Питание (V) — соедините с рабочим напряжением микроконтроллера.
  • Земля (G) — соедините с пином GND микроконтроллера.

Вторая группа

  • Сигнальный (RX) — цифровой выход Wi-Fi модуля. Используется для приёма данных из микроконтроллера. Подключите к пину TX микроконтроллера.
  • Сигнальный (P) — используйте для перевода модуля в режим сна. Подключите к любому цифровому пину микроконтроллера.

При подключении к Arduino или Iskra используйте Troyka Shield. Чтобы полностью избавиться от проводов, возьмите Troyka Slot Shield.

Характеристики

Общие

  • Модуль: ESP-12 с чипом ESP8266EX
  • Выходной интерфейс: UART
  • Объём Flash-памяти: 2 МБ
  • Беспроводной интерфейс: Wi-Fi 802.11 b/g/n 2,4 ГГц
  • Режимы работы:
    • Клиент (STA)
    • Точка доступа (AP)
    • Клиент + Точка доступа (STA + AP)
  • Напряжение питания: 3,3–5 В
  • Потребляемый ток: до 250 мА
  • Габариты: 50,8×25,4 мм

В режиме самостоятельного контроллера

  • Тактовая частота: 80 МГц
  • Портов ввода-вывода всего: 5
  • Портов с ШИМ: 5 (программный)
  • Разрядность ШИМ: по умолчанию 10 бит
  • Максимальный ток с пина или на пин: 12 мА
