Raspberry Pi Zero 2 W — це мініатюрний комп'ютер з процесором Broadcom BCM2835 на 1 ГГц, 512 МБ RAM та модулями Wi-Fi / Bluetooth, який повноцінно замінить ПК або ноутбук. Платформа підійде не тільки для веб-серфінгу всесвітньої павутини та спілкування в соцмережах, але й стане «мозком» для робота чи розумного дому. На борту малини помістилися процесор Broadcom BCM2835, модуль оперативної пам'яті об'ємом 512 МБ, бездротовий модуль Wi-Fi/Bluetooth, слот для microSD-карти, роз'єм HDMI Mini, конектор для штатних камер CSI Mini і два порти USB Micro.
