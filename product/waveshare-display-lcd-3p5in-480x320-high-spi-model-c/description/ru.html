Общие сведения

Используйте цветной дисплей Waveshare LCD 3,5” (480×320 / HIGH SPI / Model C) для вывода изображения с Raspberry Pi, Orange Pi, Rock Pi и других одноплатных ПК со стандартным GPIO-разъёмом на 40 контактов. Дисплейный модуль пригодится для просмотра медиафайлов, прохождения видеоигр или трансляции видео с камеры.

Встроенная резистивная сенсорная панель позволяет задействовать экран в роли планшета для удобного управления системой без клавиатуры и мыши. Резистивный тачскрин нужно программно откалибровать для более точной и корректной работы.

Начинка

Дисплейный модуль Waveshare LCD 3,5” Model C выполнен на матрице TFT TN с диагональю 3,5 дюймов, разрешением 480×320 пикселей и отображением 262144 цветов. Резистивная сенсорная панель реагирует на любое нажатие, включая пальцы в перчатках и стилусы без специального покрытия.

Экран оснащён встроенной светодиодной подсветкой, обеспечивающей видимость изображения при любых условиях освещённости.

Подключение

Дисплей предусмотрен для установки на миникомпьютеры форм-фактора Raspberry Pi через GPIO-колодку без пайки и проводов. Для получения видеосигнала используется интерфейс High-Speed SPI 125 МГц, а контакты GPIO передают данные с сенсорной панели по интерфейсу SPI.

Питание

Плата с дисплеем питается от Raspberry Pi через GPIO-разъём напряжением 5 вольт с потреблением тока до 200 мА.
