Общие сведения

Объектив OpenMV Telephoto Lens для камер OpenMV с четырёхкратным зумом и байонетом M12/0,5 мм позволяет снимать удалённые объекты, которые выглядят слишком мелкими для штатного объектива. Телеобъектив идеально подходит для ситуаций, когда требуется исследовать объекты крупным планом. Например, для системы трекинга глазных зрачков без использования ИК-подсветки.

Комплектация

  • 1× Объектив OpenMV Telephoto Lens
  • 1× Защитный колпачок

Характеристики

  • Совместимость: камеры OpenMV
  • Байонет: M12/0,5 мм
  • Фокусное расстояние: 12 мм
  • Эквивалентный зум: 4×
  • Диафрагма: F2.0
  • Угол обзора по горизонтали: 18,8°
  • Угол обзора по вертикали: 14°
  • ИК-фильтр: 650 нм (убираемый)
  • Размеры: 14×14×15 мм
  • Вес: 4 г
