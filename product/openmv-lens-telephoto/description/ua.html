Загальна інформація

Використовуйте об'єктив OpenMV Telephoto Lens з чотирикратним зумом для моніторингу віддалених об'єктів, які важко розглянути через стандартний об'єктив. Лінза ідеально підходить для аналізу об'єктів на великих відстанях, зйомки деталей з дрону у важкодоступних місцях та спостереження за дикою природою на відстані без занепокоєння тварин.

Об'єктив виконаний з байонетом M12/0,5 мм та чудово підійде для камер машинного зору OpenMV.

  • 1× Захисний ковпачок
