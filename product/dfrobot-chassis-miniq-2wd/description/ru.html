Используйте роботизированную платформу DFRobot miniQ 2WD в качестве шасси для создания мобильного робота.

Платформа DFRobot miniQ 2WD — это сет компонентов для самостоятельной сборки автономного бота. Основание робоплатформы выполнено в виде перфорированного плоского диска. Круглая форма шасси способствует удобному маневрированию вокруг препятствий и минимизирует вероятность застревания робота. За движение отвечает два коллекторных мотора с металлическими редукторами, которые пропорционально обеспечивают скорость и крутящий момент. На выходном валу каждого из двигателей закреплено колесо с резиновым протектором, обеспечивающее надёжное сцепление с любой поверхностью. В качестве балансирующих элементов на платформе установлены две шаровые опоры, которые позволяют роботу поддерживать равновесие.

Платформа miniQ 2WD — это только корпус с механическими элементами. Чтобы оживить робота добавьте к шасси контроллер, драйвер моторов и другую сопутствующую электронику. На платформе предусмотрены монтажные отверстия для крепления датчиков, модулей и другой электроники.
