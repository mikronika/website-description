Силовой ключ из линейки Troyka-модулей подойдёт для коммутации цепей с большим постоянным током. Модуль пригодится в создании автополива для управления помпой, оформлении помещений для установки яркости светодиодной ленты и в охранных системах для открытия/закрытия электронных замков. Вам не придётся возиться с пайкой, макетной платой и необходимой обвязкой для работы полевого транзистора — всё уже готово.
