Силовий ключ з лінійки Troyka-модулів підійде для комутації ланцюгів з великим постійним струмом. Модуль стане в нагоді у створенні автополиву для управління помпою, оформленні приміщень для встановлення яскравості світлодіодної стрічки та в охоронних системах для відкриття/закриття електронних замків. Вам не доведеться поратися з паянням, макетною платою та необхідною обв'язкою для роботи польового транзистора — все вже готове.
