Хотите начать заниматься программированием микроконтроллеров, но сомневаетесь в выборе между Arduino или Raspberry Pi? Тогда обратите внимания на платформу Raspberry Pi Pico, которая является золотой серединой и воплотит ваши DIY-проекты в жизнь. Используйте платформу Raspberry Pi Pico для создания электронных гаджетов, метеостанций, роботов и других изобретений. Плата программируется на языке MicroPython или C++ и отлично подойдёт как начинающим мейкерам, так и опытным разработчикам.

Видеообзор

Что такое Raspberry Pi Pico?

Raspberry Pi Pico — это инновационная платформа для разработки от компании Raspberry Pi Foundation. Плата выполнена на чипе собственного производства RP2040, который станет подходящим «мозгом» для ваших гаджетов, роботов и других изобретений, где важен баланс производительности с низким энергопотреблением.

Платформа Pico работает без операционной системы: все ресурсы платформы посвящены выполнению одной программы, которая дирижирует датчиками, модулями и исполнительными устройствами.

Мы предлагаем Raspberry Pi Pico в двух вариантах: с ногами и без ног. Версия без ножек пригодится для мобильных проектов, где каждый миллиметр на счету. Для прототипирования на макетной плате подходит модификация Raspberry Pi Pico с ногами.

Начинка и особенности платы

  • Чип RP2040 собственной разработки, выполненный по технологии SoC (англ. System-on-a-Chip — система на кристалле) от компании Raspberry Pi Foundation в Великобритании
  • Двухъядерный процессор Arm Cortex M0+, с частотой до 133 МГц и RAM-памятью на 264 КБ
  • Внешняя Flash-память на 2 МБ
  • USB 1.1 с поддержкой устройства и хоста
  • Режимы сна и ожидания с низким энергопотреблением
  • 26 контактов ввода-вывода GPIO
  • 2× SPI, 2× I²C, 2× UART, 3× АЦП, 16× ШИМ
  • Встроенные часы реального времени и датчик температуры в микроконтроллер
  • 8 механизмов программируемого ввода-вывода (PIO) для поддержки настраиваемых периферийных устройств
  • Универсальное питание от 1,8 до 5,5 В

Raspberry Pi Pico vs Arduino Uno vs Raspberry Pi 4

Платформа Pico по своей архитектуре ближе к микроконтроллерам Arduino, чем к одноплатным компьютерам Raspberry Pi. Но для наглядности давайте сравним Pico с популярной платформой из мира Arduino — Arduino Uno, а из сообщества Raspberry Pi — Raspberry Pi 4.

Питание

Для питания платформы используйте разъём micro-USB или контакт VSYS. Источник определяется автоматически.

  • При питании через USB понадобится зарядник на 5 В совместно с кабелем USB (A — USB micro).
  • При питании через пин VSYS — используйте источник с выходным напряжением от 1,8 до 5,5 В. На плате расположен универсальный преобразователь питания, который при низком входном напряжении поднимет питание до 3,3 вольт, а при высоком — понизит до 3,3 вольт. В итоге вы можете использовать широкий выбор источников питания.

Программирование

Контроллер программируется по USB с помощью SDK на C/C++ или официального порта MicroPython. Поддержка Raspberry Pi Pico также появится в будущих версиях Arduino IDE.

Комплектация

  • 1× Raspberry Pi Pico (с ногами)

Характеристики

  • Модель: Raspberry Pi Pico (с ногами)
  • Микроконтроллер: RP2040
  • Количество ядер: 2× ARM Cortex-M0+ (32 бита)
  • Тактовая частота: 133 МГц
  • Flash-память: 2 МБ
  • RAM-память: 264 КБ
  • Контакты ввода-вывода GPIO: 26
  • Контакты с АЦП: 3
  • Разрядность АЦП: 12 бит
  • Контакты с ШИМ: 16
  • Каналы DMA: 12
  • Аппаратные интерфейсы: 2× UART, 2× I²C, 2× SPI
  • Встроенные периферийные устройства: Часы реального времени (RTC), Датчик температуры
  • Напряжение логических уровней: 3,3 В
  • Входное напряжение питания:
    • Через USB: 5 В
    • Через пин VSYS: 1,8–5,5 В
  • Максимальный ток потребления: 140 мА
  • Габариты: 52×21×4 мм
