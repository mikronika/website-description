Загальна інформація

Не вистачає пам'яті та обчислювальних ресурсів Arduino? Додайте потужності у свою розробку — використовуйте платформу Teensy 4.1 Ethernet з мікроконтролером NXP MIMXRT1062 на базі ядра ARM Cortex-M7 з тактовою частотою до 600 МГц. Контролер Teensy з легкістю справляється з ресурсоємними завданнями — шифруванням даних, відтворенням аудіофайлів, управлінням багатосуглобовими роботами, стабілізацією безпілотників та виконанням алгоритмів машинного навчання.

Плата розробки Teensy 4.1 підтримує програмування на C++, MicroPython та CircuitPython — обирайте зручну мову залежно від завдань та досвіду.

Особливості

  • Компактний форм-фактор для вбудовування у проекти з обмеженим простором.
  • Мікроконтролер NXP MIMXRT1062 на ядрі ARM Cortex-M7 з тактовою частотою до 600 МГц
  • Підтримка мов програмування: C++, MicroPython чи CircuitPython.
  • Вбудований USB-порт для програмування та обміну даними.

Начинка

Розглянемо основні можливості плати Teensy 4.1 Ethernet.

Основний мікроконтролер MIMXRT1062

Платформа Teensy 4.1 виконана на мікроконтролері NXP MIMXRT1062 з обчислювальним ядром ARM Cortex-M7. Контролер працює на тактовій частоті 600 МГц і оснащений 1 МБ SRAM-пам'яті для зберігання змінних.

Додатковий мікроконтролер GD32E230F8

Щоб не займати жодного байта пам'яті основного контролера MIMXRT1062, для зберігання та запису завантажувача використовується додатковий мікроконтролер GD32E230F8.

Flash-пам'ять 2 МБ

Прошивка користувача зберігається на окремій мікросхемі зовнішньої Flash-пам'яті Winbond W25Q64JV на 8 МБ.

Чіп Ethernet DP83825

Чіп DP83825 дозволяє платформі Teensy 4.1 Ethernet виходити до інтернету та створювати проєкти на базі інтернет речей IoT. Для підключення кабелю витої пари RJ45 потрібен перехідник Ethernet для Teensy.

DC-DC перетворювач

На платі встановлений понижуючий DC-DC перетворювач, який забезпечує живлення мікроконтролера та іншої логіки плати. Вихідне напруження 3,3 В з максимальним вихідним струмом 250 мА.

Живлення

Для живлення платформи Teensy 4.1 Ethernet можна використовувати порт USB або контакт Vin.

  • При живленні через USB — використовуйте порт ПК або зарядний пристрій на 5 В.
  • При живленні через пін Vin — використовуйте джерело з вихідним напруженням від 3,3 до 5 В.

Програмування

Для зв'язку платформи Teensy 4.1 Ethernet з комп'ютером використовуйте кабель USB Micro.

Плата Teensy 4.1 підтримує наступні інструменти програмування:

  • Arduino IDE з патчем Teensyduino — середовище розробки з кодом на C++.
  • Thonny Python IDE — середовище розробки з кодом на MicroPython.
  • Mu Editor — середовище розробки з кодом на CirclePython від Adafruit.

Всі подробиці по підключенню, налаштуванню та програмуванню платформи Teensy 4.1 Ethernet читайте в керівництві на сайті виробника.

Розпіновка

На платі розміщено 65 контактних отворів та 18 контактних майданчиків SMT для підключення живлення, додаткових модулів та сенсорів.

  • Загалом фізичних контактів: 83 (65 + 18)
  • Контактних отворів: 65
  • Контактних майданчиків: 18
  • Контакти вводу-виводу GPIO: 55
  • Контакти з ШІМ: 35
  • Контакти з АЦП: 18
  • Апаратні інтерфейси: 8× UART, 3× SPI, 3× I²C, 3× CAN, 2× I²S, 1× S/PDIF

Напруга логічних рівнів виводів контролера Teensy 4.1 становить 3,3 В. Підключення периферії з логічним рівнем 5 В призведе до пошкодження платформи.
