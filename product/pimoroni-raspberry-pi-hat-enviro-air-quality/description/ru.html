Используйте метеокомплекс Pimoroni Enviro+ с миникомьютером Raspberry Pi для отслеживания климатических параметров, качества воздуха, уровней освещённости и интенсивности окружающего шума. Для наглядной индикации показаний на Enviro+ установлен миниатюрный цветной IPS-дисплей с диагональю 0,96” и разрешением 160×80.

Метеостанция Enviro+ Air Quality содержит комплекс датчиков для различных наблюдений:

  • Климатический сенсор BME280 — измеряет температуру, влажность воздуха и атмосферное давление.
  • Датчик освещённости и приближения LTR-559 — определяет уровень освещенности и фиксирует приближение объектов. Сенсор также можно использовать в роли бесконтактной кнопки для переключения данных на встроенном экране.
  • МЭМС-микрофон SPU0410HR5H-PB — выступает в роли датчика интенсивности окружающего шума.
  • Датчик газа MiCS-6814 — позволяет оценить качество воздуха, анализируя общее количество посторонних примесей. Сенсор состоит из трёх чувствительных элементов, которые реагируют на различные группы летучих веществ:
    • Угарный газ (CO) и восстановительные газы.
    • Диоксид азота (NO2) и прочие окислители.
    • Аммиак (NH3).

Метеостанция Pimoroni Enviro+ предусмотрена для установки на миникомпьютеры Raspberry Pi через GPIO-разъём без пайки и проводов.
