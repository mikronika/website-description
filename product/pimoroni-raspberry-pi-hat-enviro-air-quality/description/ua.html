Використовуйте метеокомплекс Pimoroni Enviro+ з мінікомп'ютером Raspberry Pi для відстеження кліматичних параметрів, якості повітря, рівнів освітленості та інтенсивності навколишнього шуму. Для наочної індикації показників на Enviro+ встановлений мініатюрний кольоровий IPS-дисплей з діагоналлю 0,96” та роздільною здатністю 160×80.

Метеостанція Enviro+ Air Quality містить комплекс датчиків для різних спостережень:

  • Кліматичний сенсор BME280 — вимірює температуру, вологість повітря та атмосферний тиск.
  • Датчик освітленості та наближення LTR-559 — визначає рівень освітленості та фіксує наближення об'єктів. Сенсор також можна використовувати як безконтактну кнопку для перемикання даних на вбудованому екрані.
  • МЕМС-мікрофон SPU0410HR5H-PB — виконує функцію датчика інтенсивності навколишнього шуму.
  • Датчик газу MiCS-6814 — дозволяє оцінити якість повітря, аналізуючи загальну кількість сторонніх домішок. Сенсор складається з трьох чутливих елементів, які реагують на різні групи летючих речовин:
    • Окис вуглецю (CO) та відновлювальні гази.
    • Діоксид азоту (NO2) та інші окислювачі.
    • Аміак (NH3).

Метеостанція Pimoroni Enviro+ призначена для встановлення на мінікомп'ютери Raspberry Pi через GPIO-роз'єм без пайки та проводів.
