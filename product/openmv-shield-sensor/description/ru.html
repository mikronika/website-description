Общие сведения

Используйте интерфейсную плату OpenMV Sensor Shield в качестве универсального хаба для быстрого и удобного подключения датчиков, модулей и сервоприводов к камерам машинного зрения OpenMV. Это позволяет избежать пайки, разрезания проводов и использования самодельных переходников.

Начинка

На плате расширения OpenMV Sensor Shield выведены разъёмы:

  • 3× Контакты S-V-G для подключения сервоприводов.
  • 10× Контакты S-V-G для подключения датчиков и модулей

Подключение

OpenMV Sensor Shield устанавливается на контактные колодки камеры OpenMV. Интерфейсная плата дублирует сигнальные линии с камеры машинного зрения в удобные разъёмы S-V-G.

Питание

Модуль получает питание напрямую от камеры OpenMV. Для работы с сервоприводами, требующими дополнительного питания, предусмотрен разъём DC Barrel Jack для подключения внешнего источника.

Комплектация

  • 1× Плата расширения OpenMV Sensor Shield

Характеристики

  • Модель: Интерфейсная плата OpenMV Sensor Shield
  • Совместимость: камеры OpenMV
  • Физический интерфейс: контактные колодки OpenMV
  • Напряжение питания: 3,3 В от камеры OpenMV
  • Размеры: 36×27×20 мм
  • Вес: 8 г
