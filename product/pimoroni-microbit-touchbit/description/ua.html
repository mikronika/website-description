Pimoroni Touch:bit — це компактна сенсорна панель на шість кнопок для контролерів BBC micro:bit. Клавіатура стане в нагоді як пульт управління для запуску різних подій. Наприклад, як геймпад для ігор на світлодіодній матриці або панель вводу для взаємодії з елементами розумного дому.

За зчитування натискань відповідає чіп CAP1166, який забезпечує оптимальну чутливість сенсорних майданчиків. На кожній кнопці передбачено білий світлодіод у якості індикатора дотику.

Сенсорна клавіатура Pimoroni Touch:bit розроблена для встановлення на контролери BBC micro:bit через крайовий роз'єм без пайки та проводів. Для комунікації використовується програмний інтерфейс I²C.
