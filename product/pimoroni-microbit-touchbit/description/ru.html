Pimoroni Touch:bit — это компактная сенсорная панель на шесть кнопок к контроллерам BBC micro:bit. Клавиатура пригодится в качестве пульта управления для запуска различных событий. Например, в роли геймпада для игр на светодиодной матрице или панели ввода для воздействия на элементы умного дома.

За считывания нажатий отвечает чип CAP1166, обеспечивающий оптимальную чувствительность сенсорных площадок. На каждой кнопке предусмотрен белый светодиод в роли индикатора касания.

Сенсорная клавиатура Pimoroni Touch:bit заточена для установки на контроллеры BBC micro:bit через краевой разъём без пайки и проводов. Для коммуникации используется программный интерфейс I²C.
