Общие сведения

Объектив OpenMV IR Lens для камер OpenMV с байонетом M12/0,5 мм позволяет расширить спектр изображения, обеспечивая возможность видеть инфракрасные источники света, которые недоступны для стандартных объективов с ИК-фильтром на 650 нм. Линза идеально подходит для ситуаций, где требуется отслеживание инфракрасных меток. Например, для системы трекинга глаз, где движение зрачков можно отслеживать по отражённому свету в ИК-диапазоне.

Комплектация

  • 1× Объектив OpenMV IR Lens
  • 1× Защитный колпачок

Характеристики

  • Совместимость: камеры OpenMV
  • Байонет: M12/0,5 мм
  • Фокусное расстояние: 2,8 мм
  • Диафрагма: F2.0
  • Угол обзора по горизонтали: 70,8°
  • Угол обзора по вертикали: 55,6°
  • Размеры: 14×14×17 мм
  • Вес: 5 г
