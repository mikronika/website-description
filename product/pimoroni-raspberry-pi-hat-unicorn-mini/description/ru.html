Pimoroni HAT Unicorn Mini — это светодиодный дисплей к миникомпьютерам Raspberry Pi с разрешением 17×7 точек в виде 119 ярких цветных светодиодов. Матричный экран пригодится для вывода текста, анимации и примитивной графики.

Матрица из светодиодов подключена к двум чипам Holtek HT16D35A, которые выполняют роль драйвера между экраном и внешними управляющими платами. Микросхемы также позволяют устанавливать цвет и регулировать яркость. Для вспомогательного управления событиями на матрице расположены четыре тактовых кнопки.

Pimoroni HAT Unicorn Mini предусмотрена для установки на миникомпьютеры Raspberry Pi через GPIO-разъём без пайки и проводов.
