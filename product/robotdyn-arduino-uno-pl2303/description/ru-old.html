Arduino Uno PL2303 — высококачественная реплика оригинальной итальянской платы от производителя RobotDyn. Платформа поможет создать различные электронные устройства под свою задумку: от банальной гирлянды до системы «умный дом». Простота и дружелюбность Arduino позволяет использовать плату от начинающего мейкера до опытного разработчика.

Общие сведения

RobotDyn Arduino Uno — это простая платформа для разработки, которая позволит собрать уникальное электронное устройство или повторить уже существующее. По функционалу плата полностью повторяет оригинальную итальянскую Arduino Uno и является более доступной.

Начинка

Платформа выполнена на базе микроконтроллера ATmega328P с тактовой частотой 16 МГц.

Процессор обладает тремя видами памяти:

  • 32 КБ Flash-памяти, из которых 0,5 КБ используются загрузчиком, который позволяет прошивать Arduino Uno с обычного компьютера через USB. Flash-память постоянна и её предназначение — хранение программ и сопутствующих статичных ресурсов.
  • 2 КБ SRAM-памяти, которые предназначены для хранения временных данных, например переменных программы. По сути, это оперативная память платформы. SRAM-память энергозависимая, при выключении питания все данные сотрутся.
  • 1 КБ энергонезависимой EEPROM-памяти для долговременного хранения данных, которые не стираются при выключении контроллера. По своему назначению — это аналог жёсткого диска для Arduino.

Для связи с компьютером используется USB-UART преобразователь на чипе PL2303 от компании Prolific Technology.

Распиновка

На плате расположены 20 контактов ввода/вывода для подключения внешних устройств:

  • Шесть пинов позволяют представить аналоговое напряжение в цифровом виде.
  • Шесть пинов позволяют из цифровых значений вывести аналоговое значение в виде ШИМ-сигнала.
  • Аппаратные интерфейсы: 1× SPI, 1× I²C, 1× UART.

Программирование

Для программирования платформы Arduino Uno используйте:

  • Среда разработки Arduino IDE.
  • Визуальная среда разработки XOD IDE.
  • Произвольный инструментарий C/C++.

Микроконтроллер ATmega328P не содержит USB, поэтому для прошивки и коммуникации с ПК на плате присутствует USB-UART преобразователь PL2303. Возможно понадобится драйвер от производителя.

Питание платформы

Для питания платформы можно использовать порт USB, контакт Vin или разъём DC Barrel Jack. Источник определяется автоматически.

  • При питании через USB используйте зарядник на 5 В совместно с кабелем USB (A — B).
  • При питании через пин Vin или разъём DC Barrel Jack — используйте источник с выходным напряжением от 7 до 12 В.
