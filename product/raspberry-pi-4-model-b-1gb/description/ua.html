Мініатюрний комп'ютер Raspberry Pi 4 Model B — це повноцінна заміна ПК або ноутбуку. Платформа підійде не тільки для веб-серфінгу всесвітньої павутини та спілкування в соцмережах, але й стане «мозком» для робота чи розумного дому. На борту малини вже є чотири порти USB, два HDMI Micro виходи під монітори, бездротовий модуль Wi-Fi / Bluetooth та повноцінний Ethernet зі швидкістю до 1 Гбіт.
