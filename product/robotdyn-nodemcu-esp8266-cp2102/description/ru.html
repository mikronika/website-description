Платформа на чипе ESP8266 с Wi-Fi для разработки электронных устройств. Используйте платформу NodeMCU для создания различных сетевых проектов «Интернета вещей» с использованием беспроводной передачи данных по Wi-Fi. Плата поможет управлять домашней техникой, бытовыми приборами, мобильными роботами и проверять со своего смартфона состояние «Умного дома».
