Платформа на чіпі ESP8266 з Wi-Fi для розробки електронних пристроїв. Використовуйте платформу NodeMCU для створення різноманітних мережевих проектів «Інтернету речей» з використанням бездротової передачі даних по Wi-Fi. Плата допоможе керувати домашньою технікою, побутовими приладами, мобільними роботами та перевіряти зі свого смартфону стан «Розумного дому».
