Используйте контроллер SparkFun LilyPad Arduino USB в качестве электронного мозга для создания умной одежды. Управляющая плата со встроенным micro-USB входом поможет собрать скатерть с индикацией температуры еды, умную перчатку для управления девайсами с помощью жестов, футболку с мигающими светодиодами в такт музыке и другие девайсы в текстильном направлении.

Контроллер LilyPad Arduino USB выполнен в виде плоского круглого диска диаметром 50 мм. По внешнему диаметру на плате расположены 11 контактов в виде лепестков, предназначенных для соединения в виде пришивания с другими устройствами при помощи токопроводящей нити.

Платформа выполнена на микроконтроллере ATmega32U4 c тактовой частотой 16 МГц и программируется на языке C++. Для загрузки прошивки на плате расположен порт micro-USB.

Для питания контроллера используйте порт micro-USB или контакты-лепестки с метками «+» и «−». При подключении питания соблюдайте полярность, чтобы избежать повреждения устройства.
