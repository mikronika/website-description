Використовуйте плату розширення SparkFun Micro:bit Breakout як перехідник для встановлення контролера BBC micro:bit на макетку для подальшої сумісності зі звичними в DIY-світі датчиками та модулями. Для підключення контролера на адаптері передбачено гніздо для крайового роз'єму BBC micro:bit — ніяких проводів і тим більше пайки не знадобиться. А для комунікації з макетною платою на перехіднику розташована гребінка PLS на 20 контактів.
