Используйте плату расширения SparkFun Micro:bit Breakout в роли переходника для установки контроллера BBC micro:bit на макетку для дальнейшей совместимости с привычными в DIY-мире датчиками и модулями. Для подключения контроллера на адаптере предусмотрено гнездо для краевого разъёма BBC micro:bit — никаких проводов и тем более пайки не понадобится. А для коммуникации с макетной платой на переходнике расположена гребёнка PLS на 20 контактов.
