RobotDyn преобразователь интерфейсов USB-UART

Как прошить плату без собственного USB-порта? Используйте конвертер USB-UART, который поможет подключить различные устройства к USB-входу компьютера для дальнейшей прошивки или отладки.

Общие сведения

Преобразователь USB-UART от производителя RobotDyn выступает мостом между USB-портом компьютера и микроконтроллером. Доступная цена позволяет использовать конвертер широкому кругу разработчиков электронных устройств. Также модуль USB-UART поможет подключиться к бытовым электронным девайсам для включения и расширения различных скрытых настроек, например к роутерам, видеокамерам, регистраторам и другим цифровым устройствам.

Обратите внимание! Модуль поставляется с пинами в виде лужёных отверстий под пайку. Вооружитесь паяльником, чтобы припаять штыревые разъёмы и подключить его к макетной плате, тем самым получить доступ ко всем контактам.

Начинка и принцип работы

USB-UART преобразователь основан на микросхеме CP2104 от Silicon Laboratories. Чип CP2104 имеет качественную поддержку драйверов, а значит будет стабильно работать на Windows, Linux и MacOS.

Для начала работы выполните несколько простых шагов:

  • Подключите конвертер к USB-порту компьютера.
  • Дождитесь автоматической загрузки драйверов. Если этого не произошло, установите ПО самостоятельно.
  • После завершения установки в системе появится виртуальный СОМ-порт. А это значит, что можно смело подключать платы разработки к конвертеру. Для коммуникации используйте интересующие пины на плате.

Преобразователь похвастается функцией автоматического сброса для плат Arduino: больше не нужно возиться с кнопкой Reset при прошивки устройства.

Конвертер по умолчанию использует уровни сигнала 5 В. Для использования уровня 3,3 В разомкните перемычку для 5-вольтовых паяльных площадок на обратной стороне платы и капните каплей припоя на площадках для 3,3 В.

Распиновка

На плате расположены:

  • 18 контактов ввода-вывода общего назначения.
  • Светодиод Status, который мигает при получении и передаче данных.
