Загальна інформація

Використовуйте кольоровий дисплей Waveshare LCD 5” (800×480 / HDMI / Model B) для виведення зображення з Raspberry Pi, ігрових консолей, персональних комп’ютерів та інших пристроїв із HDMI-виходом. Дисплейний модуль стане в нагоді для перегляду медіафайлів, проходження відеоігор або трансляції відео з камери.

Вбудована резистивна сенсорна панель дозволяє використовувати екран як планшет для зручного керування системою без клавіатури та миші. Резистивний тачскрін потрібно програмно відкалібрувати для більш точної та коректної роботи.

Начинка

Дисплейний модуль Waveshare LCD 5” Model B виконаний на матриці TFT TN з діагоналлю 5 дюймів, роздільною здатністю 800×480 пікселів і відображенням 262144 кольорів. Резистивна сенсорна панель реагує на будь-яке натискання, зокрема на дотики пальців у рукавичках та стилусів без спеціального покриття.

Екран оснащений вбудованою світлодіодною підсвіткою, що забезпечує видимість зображення за будь-яких умов освітлення.

Підключення

На платі дисплея розташований порт HDMI для отримання відеосигналу від зовнішніх пристроїв, а також роз'єм USB Micro для передавання даних з сенсорної панелі.

Живлення

Модуль живиться через порт USB Micro напругою 5 вольт зі споживанням струму до 385 мА. Живлення можна отримати через USB-порт від пристрою джерела відео або від зовнішнього блока живлення.
