Общие сведения

Используйте цветной дисплей Waveshare LCD 5” (800×480 / HDMI / Model B) для вывода изображения с Raspberry Pi, игровых консолей, персональных компьютеров и других устройств с HDMI-выходом. Дисплейный модуль пригодится для просмотра медиафайлов, прохождения видеоигр или трансляции видео с камеры.

Встроенная резистивная сенсорная панель позволяет задействовать экран в роли планшета для удобного управления системой без клавиатуры и мыши. Резистивный тачскрин нужно программно откалибровать для более точной и корректной работы.

Начинка

Дисплейный модуль Waveshare LCD 5” Model B выполнен на матрице TFT TN с диагональю 5 дюймов, разрешением 800×480 пикселей и отображением 262144 цветов. Резистивная сенсорная панель реагирует на любое нажатие, включая пальцы в перчатках и стилусы без специального покрытия.

Экран оснащён встроенной светодиодной подсветкой, обеспечивающей видимость изображения при любых условиях освещённости.

Подключение

На плате дисплея расположен порт HDMI для получения видеосигнала от внешних устройств, а также разъём USB Micro для передачи данных с сенсорной панели.

Питание

Модуль питается через порт USB Micro напряжением 5 вольт с потреблением тока до 385 мА. Питание можно получить через USB-порт от устройства источника видео или от внешнего блока питания.
