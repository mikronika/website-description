Общие сведения

Используйте дисплей Waveshare E-Ink 2,13” HAT (212×104 / Жёлтый) на электронных чернилах для энергоэффективного вывода информации в различных областях применения:

  • Умный дом — индикаторы температуры, погодные станции, системы управления.
  • Носимые устройства — электронные бейджи, умные часы.
  • Промышленные решения — бирки и ценники, информационные табло, приборные панели.
  • Электронные книги — небольшие ридеры, устройства для записей.
  • DIY-проекты и IoT — информационные панели, датчики, кнопки управления.

В основе работы e-Paper дисплеев лежит технология MED (Microencapsulated Electrophoretic Display), разработанная компанией E-Ink Corporation. MED работает по принципу электрофореза: внутри экрана есть крошечные капсулы, наполненные заряженными частицами чернил. Когда на них подаётся электрический сигнал, частицы перемещаются, формируя изображение. Это позволяет отображать текст и графику без постоянного потребления энергии.

Экран отличается исключительно низким энергопотреблением — энергия тратится лишь на обновление изображения. Свободные углы обзора и хорошая читаемость под прямым светом позволяют обходится без подсветки. Последнее изображение хранится даже при выключенном питании.

Модуль совместим с микроконтроллерами и одноплатными компьютерами, включая Arduino, Raspberry Pi, ESP32.

Начинка

Дисплейный модуль Waveshare E-Ink 2,13” HAT имеет диагональ 2,13 дюйма, разрешение 212×104 пикселей и угол обзора больше 170°. Изображение выводится в чёрном, белом и жёлтом цвете без полутонов. У каждого пикселя есть только два состояния: включён и выключен. При обновлении изображения на экране происходит несколько циклов мерцания — это нормально для дисплеев E-Ink.

Подключение

Дисплей предусмотрен для установки на миникомпьютеры форм-фактора Raspberry Pi через GPIO-колодку без пайки и проводов. Raspberry Pi общается с модулем по интерфейсу SPI через контакты GPIO.

Также возможно SPI-подключение к управляющим платам с помощью 8-контактного провода со следующей распиновкой:

  • VCC — питание 3,3 В;
  • GND — земля;
  • DIN — линия SPI MOSI для передачи данных от главного устройства к подчиненному;
  • CLK — линия SPI SCLK с тактовым сигналом
  • CS — выбор чипа SPI (низкий уровень — активное состояние);
  • DC — управляющий пин Data/Command (высокий уровень для данных, низкий для команд);
  • RST — инициализация сброса при низком уровне;
  • BUSY — сигнал занятого состояния при низком уровне.

Питание

Встроенный преобразователь напряжения позволяет подключать модуль к управляющим платформам с рабочим напряжением от 3,3 до 5 В.
