Общие сведения

Используйте цветной дисплей Waveshare AMOLED 5,5” (1920×1080 / HDMI) для вывода изображения с Raspberry Pi, игровых консолей, персональных компьютеров и других устройств с HDMI-выходом. Дисплейный модуль пригодится для просмотра медиафайлов, прохождения видеоигр или трансляции видео с камеры. Встроенная ёмкостная сенсорная панель позволяет задействовать экран в роли планшета для удобного управления системой без клавиатуры и мыши.

Начинка

Дисплейный модуль Waveshare AMOLED 5,5” выполнен на матрице AMOLED с диагональю 5,5 дюймов, разрешением 1920×1080 пикселей, углом обзора до 170° и отображением 16,8 миллионов цветов. Ёмкостная сенсорная панель поддерживает до 5 одновременных касаний и защищена закаленным стеклом с твердостью 6H, которое повышает устойчивость экрана к царапинам и повреждениям.

Матрица AMOLED использует технологию самосветящихся пикселей, обеспечивая чёткое изображение без необходимости во внешней подсветке. У дисплея есть пять фиксированных уровней яркости, которые переключаются нажатием кнопки на задней стороне модуля. На модуле предусмотрен аудиовыход 3,5 мм для подключения наушников.

Подключение

На плате дисплея расположен порт HDMI для получения видеосигнала от внешних устройств, а также разъём USB Micro для передачи данных с сенсорной панели.

Питание

Модуль питается через порт USB Micro напряжением 5 вольт с потреблением тока до 650 мА. Питание можно получить через USB-порт от устройства источника видео или от внешнего блока питания.
