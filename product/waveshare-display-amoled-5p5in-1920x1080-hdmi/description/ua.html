Загальна інформація

Використовуйте кольоровий дисплей Waveshare AMOLED 5,5” (1920×1080 / HDMI) для виведення зображення з Raspberry Pi, ігрових консолей, персональних комп’ютерів та інших пристроїв із HDMI-виходом. Дисплейний модуль стане в нагоді для перегляду медіафайлів, проходження відеоігор або трансляції відео з камери. Вбудована ємнісна сенсорна панель дозволяє використовувати екран як планшет для зручного керування системою без клавіатури та миші.

Начинка

Дисплейний модуль Waveshare AMOLED 5,5” виконаний на матриці AMOLED з діагоналлю 5,5 дюймів, роздільною здатністю 1920×1080 пікселів, кутом огляду до 170° і відображенням 16,8 мільйонів кольорів. Ємнісна сенсорна панель підтримує до 5 одночасних дотиків та захищена загартованим склом твердістю 6H, що підвищує стійкість екрану до подряпин і пошкоджень.

Матриця AMOLED використовує технологію самосвітних пікселів, забезпечуючи чітке зображення без необхідності у зовнішньому підсвічуванні. У дисплея є п’ять фіксованих рівнів яскравості, які перемикаються натисканням кнопки на задній стороні модуля. На модулі передбачений аудіовихід 3,5 мм для підключення навушників.

Підключення

На платі дисплея розташований порт HDMI для отримання відеосигналу від зовнішніх пристроїв, а також роз'єм USB Micro для передавання даних з сенсорної панелі.

Живлення

Модуль живиться через порт USB Micro напругою 5 вольт зі споживанням струму до 650 мА. Живлення можна отримати через USB-порт від пристрою джерела відео або від зовнішнього блока живлення.
