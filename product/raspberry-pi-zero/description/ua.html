Raspberry Pi Zero — це мініатюрний комп'ютер з процесором Broadcom BCM2835 на 1 ГГц та пам'яттю 512 МБ RAM, який повноцінно замінить ПК або ноутбук. Платформа підійде не тільки для веб-серфінгу всесвітньої павутини та спілкування в соцмережах, але й стане «мозком» для робота чи розумного дому. На борту малини помістилися процесор Broadcom BCM2835, модуль оперативної пам'яті об'ємом 512 МБ, слот для microSD-карти, роз'єм HDMI Mini, конектор для штатних камер CSI Mini та два порти USB Micro.
