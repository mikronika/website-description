Raspberry Pi Zero — полноценный одноплатный микрокомпьютер, размером с iPod, который станет достойной заменой ноутбука или ПК. На борту самой миниатюрной из «малинок» поместились ARM-процессор Broadcom BCM2835, модуль оперативной памяти объёмом 512 МБ, слот для microSD-карты, разъём mini-HDMI, коннектор для штатных камер и два порта micro-USB.
