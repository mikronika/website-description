Raspberry Pi Zero — полноценный одноплатный микрокомпьютер, размером с iPod, станет достойной заменой ноутбука или ПК. На борту самой миниатюрной из «малинок» поместились процессор Broadcom BCM2835, модуль оперативной памяти объёмом 512 МБ, слот для microSD карты, разъём HDMI Mini, коннектор для штатных камер CSI Mini и два порта USB Micro.
