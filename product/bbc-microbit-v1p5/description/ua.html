Контролер BBC micro:bit — це компактний самодостатній освітній комплекс для вивчення основ електроніки та програмування. Плата буде цікава електронщикам, дизайнерам, програмістам та всім винахідникам, які бажають зібрати власний гаджет від простого робота до системи «Розумний дім». Навчальна платформа розроблена компанією ВВС (British Broadcasting Corporation) для проведення уроків у школах і науково-технічних гуртках з дітьми від 7 років. Контролер програмується в середовищі MakeCode на візуальних блоках або текстових мовах JavaScript та Python.

Начинка

Платформа виконана на мікроконтролері Nordic nRF51822 з обчислювальним ядром ARM Cortex-M0 на частоті 16 МГц з двома блоками пам'яті: Flash на 256 КБ та RAM на 16 КБ. Кристал також містить бездротовий модуль Bluetooth 4.1 для зв'язку по повітрю та кліматичний сенсор для визначення температури навколишнього середовища.

Головна особливість платформи BBC micro:bit — на борту є все необхідне для швидкого та комфортного старту:

  • Допоміжний чіп NXP KL26 для прошивки та обміну даними через USB.
  • Світлодіодна панель у вигляді матриці 5×5 з 25 яскравих червоних точок для виведення бігучої строки, анімацій та графіки. Дев'ять світлодіодів з матриці також використовуються «хитрим чином», як сенсор рівня освітленості.
  • Акселерометр NXP MMA8653 вимірює прискорення для визначення нахилу плати у просторі, використовується у крокомірах та реагує на струшування.
  • Магнітометр NXP MAG3110 фіксує магнітне поле для пошуку прихованої проводки, визначення напрямку на північ у ролі компасу та фіксації магнітних аномалій.
  • Дві тактові кнопки A та B для користувача.

Живлення

Для живлення платформи BBC micro:bit v1.5 можна використовувати порт USB, роз'єм JST PH-2 або контакт 3V. Джерело визначається автоматично.

  • При живленні через USB — використовуйте джерело з вихідною напругою 5 В. Наприклад, блок живлення на 5 В з USB-виходом і кабель USB Micro.
  • При живленні через контакт 3V — використовуйте джерело з вихідною напругою 3,3 В.
  • При живленні через роз'єм JST PH-2 — використовуйте джерело з вихідною напругою від 2,1 до 3,6 В. Наприклад, батарейний відсік АА / ААА з елементами живлення та конектором JST PH-2 з проводами.

Програмування

Завантажувати прошивку у BBC micro:bit можна двома способами:

  • По дротовому підключенню через ПК
  • По бездротовому підключенню через смартфон

Розподіл контактів

На платі micro:bit v1.5 розташований краєвий роз'єм на 25 контактів для підключення живлення та додаткових модулів. Краєвий конектор містить 5 широких майданчиків з кільцями та 20 вузьких майданчиків.

  • Широкі майданчики з отворами дозволяють підключати модулі за допомогою зажимів «крокодил» або штекерів «банан» діаметром 4 мм.
  • Вузькі майданчики дозволяють підключати модулі тільки через спеціальні перехідники.
