Загальна інформація

Використовуйте кольоровий дисплей Waveshare LCD 3,5” (480×320 / HDMI / Model C) для виведення зображення з Raspberry Pi, Orange Pi, Rock Pi та інших одноплатних ПК зі стандартним GPIO-роз'ємом на 40 контактів та HDMI-виходом. Дисплейний модуль стане в нагоді для перегляду медіафайлів, проходження відеоігор або трансляції відео з камери. Вбудована резистивна сенсорна панель дозволяє використовувати екран як планшет для зручного керування системою без клавіатури та миші.

Начинка

Дисплейний модуль Waveshare LCD 3,5” Model C виконаний на матриці TFT IPS з діагоналлю 3,5 дюйма, роздільною здатністю 480×320 пікселів, кутом огляду до 160° і відображенням 262144 кольорів. Резистивна сенсорна панель реагує на будь-яке натискання, зокрема на дотики пальців у рукавичках та стилусів без спеціального покриття.

Екран оснащений вбудованою світлодіодною підсвіткою, що забезпечує видимість зображення за будь-яких умов освітлення. У дисплеї є вбудоване меню для налаштування яскравості, контрастності, рівня підсвітки та інших параметрів зображення. На модулі передбачений аудіовихід 3,5 мм для підключення навушників.

Підключення

Дисплей передбачений для встановлення на мінікомп'ютери форм-фактора Raspberry Pi через GPIO-колодку без пайки та проводів. Для отримання відеосигналу використовується інтерфейс HDMI, а контакти GPIO передають дані з сенсорної панелі через інтерфейс SPI.

Живлення

Плата з дисплеєм живиться від Raspberry Pi через GPIO-роз'єм або через HDMI-роз'єм напругою 5 вольт зі споживанням струму до 200 мА.
