Видеообзор

Общие сведения

Датчик построен на базе полупроводникового газоанализатора MQ-135. На выходе сенсора аналоговый сигнал, пропорциональный содержанию газов в окружающей среде.

Внутри датчика расположен чувствительный элемент, который состоит из керамической трубки с покрытием Al2O3 (оксид алюминия) и нанесенного на неё чувствительного слоя SnO2 (диоксида олова). Внутри трубки проходит нагревательный элемент, который нагревает чувствительный слой до температуры, при которой он начинает реагировать на определяемый газ. Чувствительность к разным газам достигается варьированием состава примесей в чувствительном слое.

Особенности

  • Форм-фактор Troyka — практичное решение для начинающих и опытных мейкеров.
  • Чувствительность ко всем вредным газам.
  • Возможность отдельно управлять нагревателем.

Подключение

Датчик газа подключается к управляющей электронике двумя способами:

  1. Штатный способ. Если нет нужды в управлении нагревателем датчика. Для коммуникации используйте только первую группу Troyka-контактов.
  2. Расширенный способ. Если есть нужда в управлении нагревателем. Для коммуникации используйте первую и вторую группу Troyka-контактов.

Первая группа

Контакт Функция Подключение
S Выходной сигнал сенсора Подключите к пину ввода-вывода микроконтроллера с поддержкой АЦП.
V Питание Подключите к питанию микроконтроллера.
G Земля Подключите к земле микроконтроллера.

Вторая группа

Контакт Функция Подключение
E Управление питанием нагревателя Подключите к пину ввода-вывода микроконтроллера с поддержкой АЦП.
H Питание нагревателя Подключите к питанию 5 вольт.
G Земля Подключите к земле микроконтроллера.

При подключении к платформам форм-фактора Arduino R3 рекомендуем использовать Troyka Shield или Troyka Slot Shield. А для компьютеров Raspberry Pi предусмотрен Troyka HAT, который подарит малине пины с АЦП. Также модуль физически совместим с breadboard’ом.

Программирование

Датчик газа MQ-135 для связи с микроконтроллером использует два аналоговых сигнала: один — для считывания концентрации газа, второй — для управления нагревателем. Для программной работы с модулем используйте библиотеки, которые облегчают составления кода программы.
