Общие сведения

Беспроводной приёмник MX–RF–5V — ресивер для приёма данных по воздуху на частоте 433 МГц. Дальность радиосигнала достигает 100 м в зоне прямой видимости, а в условиях квартиры при окружении бетона и кирпича — снижается к диапазону от 10 до 50 м. Однако сигнала будет достаточно для управления бытовыми устройствами в домашних условиях. Радиус приёма данных остаётся уверенным между самыми удалёнными уголками квартиры, даже при наличии по пути нескольких стен.

Для увеличения диапазона радиосигнала используйте антенну, которую можно изготовить самостоятельно:

  1. Отрежьте провод длинной 17 см.
  2. Накрутите кабель на цилиндрический предмет: например на шариковую ручку или карандаш.
  3. Припаяйте один конец провода к контакту ANT.

Беспроводной модуль MX–RF–5V заточен только для приёмник данных. Для передачи радиосигнала используйте беспроводной передатчик FS1000A на 433 МГц.
Приёмник обладает автоматической регулировкой усиления сигнала, поэтому чувствителен даже к незначительным пульсациям на шине питания: модуль принимает их за информационный сигнал. Используйте раздельное питание для управляющей платы и исполнительных устройств, которые увеличивают дребезжание.

Подключение

Модуль подключается к управляющим платам через три контакта:

  • VCC — питание передатчика. Соедините с рабочим напряжением контроллера.
  • DATA — передача данных (два дублирующих контакта). Подключите к цифровому пину микроконтроллера.
  • GND — земля. Соедините с пином GND контроллера.

Пины питания и данных выполнены в виде контактных штырьков с шагом 2,54 мм, которые позволяют вставлять приёмник в макетную плату.
