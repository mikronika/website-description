Используйте распознаватель голоса SparkFun EasyVR Shield 3 для управления умным домом, роботами и другими устройствами с помощью голосовых команд.

EasyVR Shield 3 поддерживает два метода распознавания голоса:

  • Speaker-independent (SI) — фонетический алгоритм: метод работает с конкретным языком и не зависит от голоса пользователя. В режиме предустановлено 6 словарей: английский, итальянский, немецкий, французский, испанский и японский.
  • Speaker-dependent (SD) — точный алгоритм: метод работает с любым языком, т.к. распознание речи ведётся с учетом индивидуальных особенностей голоса пользователя и никак не зависит от используемого языка.

Плата расширения EasyVR Shield 3 предусмотрена для установки на контроллеры форм-фактора Arduino Uno и общается с управляющей электроникой по интерфейсу UART. Для записи голосовых команд предусмотрен микрофон в комплекте, который подключается через разъём JST PH-2. Устройство умеет проигрывать в ответ заранее записанные аудиофрагменты, для этого необходимо подключить динамик в разъём JST PH-3 или наушники в аудиовыход 3,5 мм.
