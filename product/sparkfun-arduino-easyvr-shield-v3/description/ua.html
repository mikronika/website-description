Використовуйте розпізнавач голосу SparkFun EasyVR Shield 3 для керування розумним будинком, роботами та іншими пристроями за допомогою голосових команд.

EasyVR Shield 3 підтримує два методи розпізнавання голосу:

  • Speaker-independent (SI) — фонетичний алгоритм: метод працює з конкретною мовою та не залежить від голосу користувача. У режимі попередньо встановлено 6 словників: англійський, італійський, німецький, французький, іспанський та японський.
  • Speaker-dependent (SD) — точний алгоритм: метод працює з будь-якою мовою, оскільки розпізнавання мови здійснюється з урахуванням індивідуальних особливостей голосу користувача та жодним чином не залежить від використовуваної мови.

Плата розширення EasyVR Shield 3 передбачена для встановлення на контролери форм-фактору Arduino Uno і взаємодіє з керуючою електронікою через інтерфейс UART. Для запису голосових команд передбачений мікрофон у комплекті, який підключається через роз’єм JST PH-2. Пристрій здатний відтворювати у відповідь заздалегідь записані аудіофрагменти, для цього необхідно підключити динамік до роз’єму JST PH-3 або навушники до аудіовиходу 3,5 мм.
