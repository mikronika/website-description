Amperka GPRS Shield v2

Подарите Arduino мобильную связь для управления устройством, даже если оно находится в поле за тридевять земель. Плата расширения Amperka GPRS Shield позволит разработать девайс удалённого управления бойлером на даче, создать телефон-игрушку для своего ребёнка или мониторить данные в загородной тепличке. Главное условие — наличие мобильной связи.

Видеообзор

Общие сведения

GPRS Shield выполнен на чипе SIM900 с функциями GSM и GPRS. Плата расширения подарит вашему устройству возможности мобильного телефона:

  • Отправлять через SMS команды на устройство.
  • Запрашивать состояние или получать уведомления от него через SMS.
  • Звонить на устройство и проворачивать на нём различные сценарии.
  • Принимать звонки с устройства и прослушивать что происходит вокруг.
  • Выходить в интернет, опрашивать веб-сервисы, протоколировать данные.

Особенности

  • Совместимость со всеми платами формфактора Arduino R3.
  • Слот для полноразмерной SIM-карты вашего любимого оператора.
  • Стандартные гнёзда 3,5 мм для подключения динамика и микрофона.
  • Гнездо для батарейки CR1225, обеспечивающей работу встроенных часов реального времени.
  • SMA-коннектор для внешней антенны.
  • Возможность смены контактов управления.
Если вы не планируете использовать голосовую GSM-связь, то гарнитуру можно не подключать. Аналогично, если нет потребности в часах реального времени — батарейка не понадобится.

К SMA-разъёму можно подключить антенны на магните, липучке, большие, маленькие, с удлинителем и без. В комплект мы включили вертикальную антенну с усилением 2 дБ.

Подключение контроллера

Плата расширения GPRS Shield предусмотрена для установки на контроллеры формфактора Arduino Uno и формфактора Arduino Mega 2560. Для коммуникации с микроконтроллером используются контакты интерфейса UART с дополнительными пинами управления:

Контакт Функция
RX Линия приёма данных интерфейса UART
TX Линия отправки данных интерфейса UART
PK Программное включения модуля
ST Программное состояние модуля

Комплектация

  • 1× Плата Amperka GPRS Shield v2
  • 1× Антенна вертикальная с усилением 2 дБ
  • 4× Джампер

Характеристики

  • Модель: Amperka GPRS Shield v2 / AMP-B029
  • Чип связи: SIM900
  • Совместимость: Arduino Shield R3
  • Программный интерфейс: UART с дополнительными пинами управления
  • Входное напряжение питания Vcc: 5–12 В
  • Потребляемый ток Icc:
    • Нормальный режим: 20—500 мА
    • Поиск сети: до 2 A
    • Спящий режим: до 1,5 мА
  • Логическое напряжение уровней:
    • Вход: 3,3–5 В
    • Выход: 3,3–5 В
  • Поддержка частот: 850/1800 МГц
