Використовуйте перехідник Pimoroni HAT Hacker як проміжний елемент для паралельного підключення двох міні HAT до мінікомп'ютера Raspberry Pi.

Інтерфейсна плата Pimoroni HAT Hacker призначена для встановлення на мінікомп'ютери Raspberry Pi через GPIO-роз'єм без пайки та проводів. На виході перехідник надає одразу два GPIO-роз'єми для підключення двох розширень форм-фактора міні HAT. Плати розташовуються на одному рівні, залишаючи всі необхідні кнопки та індикатори перед очима.
