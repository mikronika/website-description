Amperka Ethernet Shield

Ethernet Shield позволит стать вашему устройству «Online», а это значит Arduino сможет стучатся во все облачные сервера. Выход в сеть позволит включать свет в комнате с телефона, следить за температурой в доме из любой точки земли, управлять устройствами через Telegram и другие возможности Internet of Things.

Видеообзор

Особенности

  • Совместимость со всеми контроллерами форм-фактора Arduino R3.
  • Плата выполнена на чипе Wiznet W5500.
  • Поддерживаемые протоколы: TCP, UDP, ICMP, IPv4, ARP, IGMP, PPPoE.
  • Скорость соединения: 10/100 Мб.

Подключение контроллера

Плата расширения Ethernet Shield предусмотрена для установки на контроллеры формфакторов Arduino Uno и Arduino Mega 2560. Для коммуникации с микроконтроллером используются контакты:

Контакт Функция
SS Выбор ведомого устройства. Используется для активации модуля на шине SPI.
MOSI Выход ведомого устройства. Используется для приёма данных из микроконтроллера.
MISO Вход ведомого устройства. Используется для передачи данных в микроконтроллер.
SCK Линия тактирования шины SPI. Используется для согласования скорости передачи.

Подключение сети

Для соединения по локальной сети используйте патч-корд или витую пару. Один конец кабеля подключите к роутеру или хабу, второй — к Ethernet Shield.

Ethernet Shield общается с микроконтроллером через интерфейс SPI. Для программной работы с платой расширения используйте библиотеки и функции, которые облегчают составления кода программы.

Альтернативы

На микроконтроллерной плате тяжело реализовывать обработку XML, JSON и раздавать страницы с динамическим содержимым — ограниченность в ресурсах даёт о себе знать. Поэтому если вам нужен проект с мощным веб-сервером или клиентом, присмотритесь к одноплатным компьютерам Raspberry Pi.
