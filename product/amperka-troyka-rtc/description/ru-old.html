Часы реального времени RTC / Troyka-модуль

Модуль часов реального времени из линейки Troyka-модулей поможет вашему устройству стать пунктуальным и выполнять задачи по расписанию. Часы высчитают время для подачи еды из автоматической кормушки для рыб, внесут в график переворачивания яйца в инкубаторе или зажгут ёлку на новый год.

Общие сведения

Часы служат для получения текущей секунды, минуты, часа, дня, месяца и года без затрат ресурсов микроконтроллера. Модуль пригодиться для создания будильников, сигнализаций и снятия показаний с датчиков по графику.

Часы реального времени (RTC — англ. Real Time Clock) выполнены на популярной микросхеме DS1307, которая занимается подсчётом времени.

На модуле расположен слот для часовой батарейки размера CR1225 на 3 вольта. Благодаря дополнительному автономному питанию от таблетки, перебои с электроснабжением не собьют вашу железку с толку. Перед началом использования текущие дата и время устанавливаются единожды, а затем они могут быть прочитаны пока жива батарейка.

Особенности

  • Форм-фактор Troyka — практичное решение для начинающих и опытных мейкеров.
  • Слот для часовой батарейки размера CR1225.
  • Время работы от батарейки: до 6 месяцев

Подключение

Часы реального времени подключаются к управляющей электронике через две группы Troyka-контактов.

Контакты питания

Контакт Функция Подключение
V Питание Подключите к питанию микроконтроллера.
G Земля Подключите к земле микроконтроллера.

Контакты данных

Контакт Функция Подключение
D Данные шины I²C Подключите к пину SDA микроконтроллера.
C Тактирование шины I²C Подключите к пину SCL микроконтроллера.

При подключении к платформам форм-фактора Arduino R3 рекомендуем использовать Troyka Shield или Troyka Slot Shield. А для компьютеров Raspberry Pi предусмотрен Troyka HAT. Также модуль физически совместим с breadboard’ом.

Программирование

Часы реального времени общаются с микроконтроллером через интерфейс I²C. Для программной работы с датчиком используйте библиотеки и функции, которые облегчают составления кода программы.

Характеристики

  • Модель: Часы реального времени / Troyka-модуль / AMP-B043
  • Чип: DS1307
  • Входное напряжение питания Vcc: 5 В
  • Входное напряжение питания батарейки: 3 В
  • Потребляемый ток батарейки: 500 нА (≈6 месяцев)
  • Форм-фактор батарейки: CR1225
  • Логическое напряжение уровней:
    • Вход: 5 В
    • Выход: 5 В
  • Аппаратный интерфейс: Troyka-контакты S-V-G
  • Программный интерфейс: I²C
  • I²C-адрес: 0x68
  • Размеры: 25,4×25,4 мм
