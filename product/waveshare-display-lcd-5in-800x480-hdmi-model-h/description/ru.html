Общие сведения

Используйте цветной дисплей Waveshare LCD 5” (800×480 / HDMI / Model H) для вывода изображения с Raspberry Pi, игровых консолей, персональных компьютеров и других устройств с HDMI-выходом. Дисплейный модуль пригодится для просмотра медиафайлов, прохождения видеоигр или трансляции видео с камеры. Встроенная ёмкостная сенсорная панель позволяет задействовать экран в роли планшета для удобного управления системой без клавиатуры и мыши.

Начинка

Дисплейный модуль Waveshare LCD 5” Model H выполнен на матрице TFT TN с диагональю 5 дюймов, разрешением 800×480 пикселей и отображением 262144 цветов. Ёмкостная сенсорная панель поддерживает до 5 одновременных касаний и защищена закаленным стеклом с твердостью 6H, которое повышает устойчивость экрана к царапинам и повреждениям.

Экран оснащён встроенной светодиодной подсветкой, обеспечивающей видимость изображения при любых условиях освещённости. В дисплее есть встроенное меню для настройки яркости, контрастности, уровня подсветки и других параметров изображения. На модуле предусмотрен аудиовыход 3,5 мм для подключения наушников и JST-разъём с усилением звукового сигнала для внешней акустики.

Подключение

На плате дисплея расположены порты HDMI и VGA для получения видеосигнала от внешних устройств, а также разъём USB Micro для передачи данных с сенсорной панели.

Питание

Модуль питается через порт USB Micro напряжением 5 вольт с потреблением тока до 500 мА. Питание можно получить через USB-порт от устройства источника видео или от внешнего блока питания.
