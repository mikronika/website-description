Общие сведения

Используйте плату связи OpenMV TV Shield для подключения камеры OpenMV к телевизорам и мониторам через композитный видеовыход. Плата расширения выводит видеоизображение с камеры напрямую на экран. Это удобно для отладки и демонстрации работы системы машинного зрения в реальном времени без использования компьютера.

Начинка

Плата OpenMV TV Shield использует встроенный чип VS23S010 для генерации аналогового видеосигнала в формате NTSC. Связь с основной камерой OpenMV осуществляется по SPI-интерфейсу, что позволяет выводить видео в реальном времени с разрешением до 352×240 пикселей и частотой до 60 Гц. Встроенная обвязка обеспечивает стабильную работу видеоинтерфейса без дополнительной настройки.

Подключение

OpenMV Wi-Fi Shield устанавливается на контактные колодки камеры OpenMV. Данные между платой расширения и камерой передаются через интерфейс SPI. Все остальные GPIO-выводы остаются свободными для подключения других устройств.

В среде разработки OpenMV IDE доступно множество готовых библиотек с примерами кода для работы с модулем TV Shield. Это позволяет быстрее запрограммировать свою систему на основе существующих скриптов, а не писать всё с нуля.

Питание

Модуль получает питание напрямую от камеры OpenMV.
