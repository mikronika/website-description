Використовуйте звукову карту Pimoroni Pirate Audio Headphone Amp для перетворення Raspberry Pi на портативний аудіоплеєр з якісним звуком! Плата розширення дозволяє відтворювати локальні треки або стрімити аудіо з хмарних сервісів.

За перетворення цифрового сигналу відповідає чіп PCM5100A з частотою дискретизації 192 кГц та глибиною 24 біта. Мікросхема приймає цифрові аудіодані від Raspberry Pi через інтерфейс I²S та перетворює їх у звичну звукову хвилю.

Для посилення вихідного сигналу використовується чіп PAM8908 з вихідною потужністю 25 мВт. Мікросхема PAM8908 отримує вхідний сигнал від PCM5100A, посилює його та передає на роз'єм для навушників.

Для індикації статусу поточного треку на платі розташований кольоровий IPS-дисплей на 1,3 дюйма, а чотири тактові кнопки необхідні для керування гучністю та відтворенням. Навушники підключаються до модуля через міні-джек 3,5 мм.

Pimoroni Pirate Audio Headphone Amp призначена для встановлення на мінікомп'ютери Raspberry Pi через GPIO-роз'єм без пайки та проводів.
