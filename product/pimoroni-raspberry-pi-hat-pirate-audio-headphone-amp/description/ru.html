Используйте звуковую карту Pimoroni Pirate Audio Headphone Amp для превращения Raspberry Pi в портативный аудиоплеер качественного звука! Плата расширения позволяет воспроизводить локальные треки или стримить аудио из облачных сервисов.

За преобразование цифрового сигнала отвечает чип PCM5100A с частотой дискретизации 192 кГц и глубиной 24 бита. Микросхема принимает цифровые аудиоданные от Raspberry Pi по интерфейсу I²S и преобразует в привычную звуковую волну.

Для усиления выходного сигнала используется чип PAM8908 с выходной мощностью 25 мВт. Микросхема PAM8908 получает входной сигнал от PCM5100A, усиливает его и передаёт на разъём для наушников.

Для индикации статуса текущего трека на плате расположен цветной IPS-дисплей на 1,3 дюйма, а четыре тактовых кнопки служат для управления громкостью и воспроизведением. Наушники подключаются к модулю через мини-джек 3,5 мм.

Pimoroni Pirate Audio Headphone Amp предусмотрена для установки на миникомпьютеры Raspberry Pi через GPIO-разъём без пайки и проводов.
