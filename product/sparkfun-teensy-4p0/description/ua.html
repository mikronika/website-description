Загальна інформація

Не вистачає пам'яті та обчислювальних ресурсів Arduino? Додайте потужності у вашу розробку — використовуйте платформу Teensy 4.0 з мікроконтролером NXP MIMXRT1062 на базі ядра ARM Cortex-M7 з тактовою частотою до 600 МГц. Контролер Teensy з легкістю справляється з ресурсоємними задачами — шифруванням даних, відтворенням аудіофайлів, керуванням багатосуглобовими роботами, стабілізацією безпілотників та виконанням алгоритмів машинного навчання.

Плата розробки Teensy 4.0 підтримує програмування на C++, MicroPython та CircuitPython — обирайте зручну мову залежно від задач і досвіду.

Особливості

  • Компактний форм-фактор для вбудовування в проекти з обмеженим простором.
  • Мікроконтролер NXP MIMXRT1062 на ядрі ARM Cortex-M7 з тактовою частотою до 600 МГц.
  • Підтримка мов програмування: C++, MicroPython чи CircuitPython.
  • Вбудований USB-порт для програмування та обміну даними.

Начинка

Розглянемо основні переваги плати Teensy 4.0.

Основний мікроконтролер MIMXRT1062

Платформа Teensy 4.0 виконана на мікроконтролері NXP MIMXRT1062 з обчислювальним ядром ARM Cortex-M7. Контролер працює на тактовій частоті 600 МГц та оснащений 1 МБ SRAM-пам'яті для зберігання змінних.

Допоміжний мікроконтролер GD32E230F8

Щоб не займати жодного байта пам'яті основного контролера MIMXRT1062, для зберігання і запису завантажувача використовується додатковий мікроконтролер GD32E230F8.

Flash-пам'ять 2 МБ

Прошивка користувача зберігається на окремій мікросхемі зовнішньої Flash-пам'яті Winbond W25Q16JV на 2 МБ.

DC-DC перетворювач

На платі встановлено понижувальний DC-DC перетворювач, який забезпечує живлення мікроконтролера та іншої логіки плати. Вихідна напруга 3,3 В з максимальним вихідним струмом 250 мА.

Живлення

Для живлення платформи Teensy 4.0 можна використовувати порт USB або контакт Vin.

  • При живленні через USB — використовуйте порт ПК або зарядний пристрій на 5 В.
  • При живленні через пін Vin — використовуйте джерело з вихідною напругою від 3,3 до 5 В.

Програмування

Для зв'язку платформи Teensy 4.0 з комп'ютером використовуйте кабель USB Micro.

Плата Teensy 4.0 підтримує наступні інструменти програмування:

  • Arduino IDE з патчем Teensyduino – середовище розробки з кодом на C++.
  • Thonny Python IDE – середовище розробки з кодом на MicroPython.
  • Mu Editor — середовище розробки з кодом на CirclePython від Adafruit.

Всі деталі щодо підключення, налаштування та програмування платформи Teensy 4.0 читайте в посібнику на сайті виробника.

Розпіновка

На платі розташовані 33 контактних отвори та 22 контактні майданчики SMT для підключення живлення, додаткових модулів і сенсорів.

  • Загальна кількість фізичних контактів: 55 (33 + 22)
  • Контактних отворів: 33
  • Контактних майданчиків: 22
  • Контакти вводу-виводу GPIO: 40
  • Контакти з ШІМ: 31
  • Контакти з АЦП: 14
  • Апаратні інтерфейси: 7× UART, 3× SPI, 3× I²C, 3× CAN, 2× I²S, 1× S/PDIF

Напруга логічних рівнів виводів контролера Teensy 4.0 становить 3,3 В. Підключення периферії з логічним рівнем 5 В призведе до пошкодження платформи.
