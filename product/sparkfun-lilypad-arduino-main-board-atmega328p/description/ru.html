Используйте контроллер SparkFun LilyPad Arduino Main Board в качестве электронного мозга для создания умной одежды. Управляющая плата поможет собрать скатерть с индикацией температуры еды, умную перчатку для управления девайсами с помощью жестов, футболку с мигающими светодиодами в такт музыке и другие девайсы в текстильном направлении.

Контроллер LilyPad Arduino Main Board выполнен в виде плоского круглого диска диаметром 50 мм. По внешнему диаметру на плате расположены 22 контакта в виде лепестков, предназначенных для соединения в виде пришивания с другими устройствами при помощи токопроводящей нити.

Платформа выполнена на микроконтроллере ATmega328P c тактовой частотой 16 МГц и программируется на языке C++. Для загрузки прошивки на плате расположены контакты PLS-6 к которым понадобится подключить внешний USB – UART преобразователь.

Для питания контроллера используйте контакты PLS-6 с коммуникацией через USB – UART преобразователь или контакты-лепестки с метками «+» и «−». При подключении питания соблюдайте полярность, чтобы избежать повреждения устройства.
