Общие сведения

Troyka Pad 1×1 — это посадочное место для быстрого и комфортного подключения одноюнитовых Troyka-модулей к Arduino или Raspberry Pi.

С двух сторон Troyka Pad-а распаяны по шесть Г-образных пинов с шагом 2,54 мм. Хабы удобно коммутировать с помощью трёхпроводных шлейфов.

Для надёжного крепления хаба предусмотрены два отверстия под винты М3. Если модуль не будут испытывать серьёзной физической нагрузки, можно обойтись и двухсторонним скотчем.
